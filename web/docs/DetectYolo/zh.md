# Detect Yolo - YOLO目标检测

**节点功能：** `Detect Yolo`节点使用YOLO模型进行目标检测。支持YOLOv5、YOLOv8、YOLOv9、YOLOv10、YOLOv11等多个版本。可以检测图像中的物体，为检测到的对象生成遮罩，并创建带有边界框和标签的可视化图像。支持阈值控制、按索引选择遮罩、可自定义的标签显示以及智能标签缩放功能。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | 必选 | IMAGE | - | - | 用于目标检测的输入图像张量 |
| `yolo_model` | 必选 | STRING | - | 可用的.pt文件 | YOLO模型文件（.pt格式），支持多版本 |
| `threshold` | 必选 | FLOAT | 0.3 | 0.0-1.0 | 检测置信度阈值 |
| `mask_index` | 必选 | STRING | "-1" | - | 遮罩选择的逗号分隔索引（-1或空值=全部） |
| `label` | 必选 | BOOLEAN | True | True/False | 是否在检测结果上显示标签 |
| `label_size` | 必选 | FLOAT | 1.0 | 0.1-5.0 | 标签大小缩放因子，控制字体大小和边框粗细 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `plot_image` | IMAGE | 带有边界框和可选标签的可视化图像 |
| `mask` | MASK | 选定检测对象的合并遮罩 |
