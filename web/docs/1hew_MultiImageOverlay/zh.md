# Multi Image Overlay - 多图层叠加

**节点功能：** `Multi Image Overlay` 按顺序依次叠加多个图层。RGBA 输入使用 alpha 通道进行正常的图层合成，RGB 输入视为不透明图层。支持动态输入端口 `image_1..image_N`、多种尺寸适应模式和背景颜色设置。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `fit_mode` | - | COMBO | `center` | `top_left` / `center` / `stretch` | 尺寸适应模式 |
| `color` | - | STRING | `1.0` | - | 背景颜色，支持多种颜色格式（如十六进制、RGB、颜色名称等） |
| `output_alpha` | - | BOOLEAN | `false` | - | 是否输出带有 alpha 通道的图像 |
| `image_1..image_N` | - | IMAGE | - | - | 动态输入的图像图层，按数字顺序依次叠加 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 叠加合成后的图像批次 |

## 功能说明

- 动态端口：以 `image_` 开头的输入端口支持自动扩展，按数字后缀顺序（`image_1` -> `image_2` -> ...）从下到上依次叠加。
- 尺寸对齐：以第一张图（`image_1`）的尺寸为基准画布尺寸。
- 适应模式：
  - `center`：居中放置，超出部分裁剪，不足部分透明。
  - `top_left`：左上角对齐放置。
  - `stretch`：拉伸至画布尺寸。
- Alpha 合成：正确处理 RGBA 图像的预乘 alpha 混合。RGB 图像视为完全不透明。
- 背景填充：当 `output_alpha=false` 时，使用指定的 `color` 填充最终图像的透明背景。

## 典型用法

- 图像合成：将带有透明背景的多个元素（如贴纸、水印、前景主体）叠加到背景图上。
- 批量叠加：支持批处理，当各图层批次大小不一致时，自动广播对齐到最大批次。

## 注意与建议

- `image_1` 决定了最终输出的尺寸，通常建议将背景图或主图连接到 `image_1`。
- 如果需要保留合成后的透明度供后续节点使用，请启用 `output_alpha`。
