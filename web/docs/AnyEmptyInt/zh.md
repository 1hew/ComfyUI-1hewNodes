# Any Empty Int - 通用空值检查（整数输出）

**节点功能：** `Any Empty Int`节点用于检查任意类型输入是否为空，根据检查结果返回自定义的整数值。支持多种数据类型的空值判断，并允许用户自定义空值和非空值对应的整数输出，为数值化的条件控制提供灵活的解决方案。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `any` | 必选 | * | - | - | 需要检查的任意类型输入值 |
| `empty` | 必选 | INT | 0 | -999999-999999 | 输入为空时返回的整数值，步长：1 |
| `not_empty` | 必选 | INT | 1 | -999999-999999 | 输入非空时返回的整数值，步长：1 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int` | INT | 根据空值检查结果返回的整数值 |
