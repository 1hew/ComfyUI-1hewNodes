# Image Batch to List - 批次拆分为单帧列表

**节点功能：** `Image Batch to List` 将多帧图像批次转换为 Python 列表，其中每个元素均为单帧批次（`B=1`），保持原有顺序与张量属性。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image_batch` | - | IMAGE | - | - | 输入图像批次（`B×H×W×C`） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image_list` | IMAGE_LIST | 单帧批次列表，顺序与输入一致 |

## 功能说明

- 确定性拆分：按索引逐帧切片，每个元素保持 `B=1` 批次。
- 空输入处理：当输入为空或 `B=0` 时返回空列表。
- 兼容性：适配接受列表输入的下游节点。

## 典型用法

- 将序列转换为列表供逐帧处理的节点使用。
- 在批次源头分支，构建逐帧工作流。

## 注意与建议

- 列表中的每个元素形状为批次（`1×H×W×C`），而非原始图像张量。
