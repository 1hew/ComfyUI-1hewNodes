# Any Switch Select - 多路选择透传（1…10）

**节点功能：** `Any Switch Select` 使用整数 `select` 在最多十个输入中选择其一并输出，同时返回实际生效的 `select` 值。实现惰性求值，仅计算被选中的输入端。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `select` | - | INT | 1 | 1-10 | 选择要输出的输入编号；`1` 对应 `input_1` |
| `input_1…input_10` | 可选 | ANY (`*`) | - | - | 备选输入；仅选中的输入被计算（惰性） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `output` | ANY (`*`) | 选中的输入值 |
| `select` | INT | 经过范围修正后实际生效的 `select` 值 |

## 功能说明

- 选择范围：`select` 会限制在 `[1, 10]`。
- 惰性端口：仅选中的输入端会被计算。
- 选择健壮：当所选输入未连接时，`output` 为 `None`，且节点不会继续等待该输入分支。
- 命名统一：节点、文件、参数都统一使用 `select` 语义。

## 典型用法

- 模式复用器：用单一 `select` 在多条子图中进行路由。
- 参数库选择：按 `select` 选中预设参数、模型或资源。
- 结合空值判定：与 `Any Empty Int` 配合，从存在性生成 `select`。

## 注意与建议

- 提供与 `select` 对应的 `input_{k}`；未提供的索引将产生 `None` 输出。
- 如果上游可能给出越界值，本节点会自动夹到有效范围。
