# Image Alpha Clean - 清理 RGBA 透明边缘杂点

**节点功能：** `Image Alpha Clean` 用于清理图像 alpha 通道中的弱透明噪点与细小孤岛，减少在 PS 等软件中添加描边/阴影时出现的边缘杂点问题。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像批次，建议使用 RGBA |
| `clean_strength` | - | COMBO | `balanced` | `soft / balanced / strong` | 清理强度预设，控制阈值与小连通域去除力度 |
| `detect_only` | - | BOOLEAN | False | - | 为 `True` 时仅检测噪点区域，不修改图像本身 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 清理后的 RGBA 图像批次（`detect_only=True` 时保持原图） |
| `noise_mask` | MASK | 检测出的噪点区域（`1` 表示疑似噪点） |

## 功能说明

- 预设驱动：仅保留少量常用参数，日常使用更高效。
- 噪点清理：去除弱透明碎点与细小孤立透明连通域。
- 边缘优化：对透明边缘执行颜色外扩，降低白边/黑边概率。
- 检测模式：先看 `noise_mask` 再决定是否实际清理。

## 典型用法

- 导出 PNG 资产后，在 PS 描边前先做透明边缘清理。
- 合成前统一 alpha 质量，减少后续边缘伪影。
- 打开 `detect_only` 做诊断，确认不会误伤主体再执行清理。

## 注意与建议

- `soft` 更保细节，`strong` 清理更激进。
- 输入为无 alpha 的 RGB 图像时，节点会按全不透明 alpha 处理。
