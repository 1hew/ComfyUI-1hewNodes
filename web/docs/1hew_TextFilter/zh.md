# Text Filter - 文本过滤

**节点功能：** `Text Filter` 节点用于解析多行文本并过滤注释内容，支持单行 `#` 注释与三引号注释块，可选移除空行，并进行必要的转义处理。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text` | 必选 | STRING | "" | 多行文本 | 待解析与过滤的文本 |
| `filter_empty_line` | 必选 | BOOLEAN | False | True/False | 在过滤后移除空行 |
| `filter_comment` | 必选 | BOOLEAN | False | True/False | 启用注释过滤（`#`、`'''`、`"""`） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `string` | STRING | 过滤后的文本字符串 |
