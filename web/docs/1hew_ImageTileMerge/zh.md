# Image Tile Merge - 兼容接缝的切片合成

**节点功能：** `Image Tile Merge` 使用 `tile_meta` 将切片重建为完整图像。通过基于位置与网格的余弦权重掩模在重叠区域进行平滑合成，可由 `blend_strength` 控制混合强度。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `tile` | - | IMAGE | - | - | 切片批次或单切片；支持 `B×H×W×3` 或 `H×W×3``、`actual_crop_size`，用于定位与裁剪对齐。
- 重叠较大时，若希望边界更清晰可适当降低 `blend_strength`；需要更柔和时提高该值。
