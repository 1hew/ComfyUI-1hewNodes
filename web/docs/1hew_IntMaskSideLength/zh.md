# Int Mask Side Length - 遮罩宽/高/最长/最短边

**节点功能：** `Int Mask Side Length` 基于输入遮罩尺寸返回一个整数：可选择最长边、最短边、宽度或高度。适合用于尺寸感知的工作流中，驱动下游参数（例如目标边长）。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | - | MASK | - | - | 输入遮罩批次（`B×H×W`）；仅使用其空间尺寸。 |
| `mode` | - | COMBO | `longest` | `longest` / `shortest` / `width` / `height` | 选择输出的边长类型。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int` | INT | 选定边长的像素值。 |

## 功能说明

- 尺寸提取：读取遮罩的 `H`、`W` 并按 `mode` 计算。
- 批次友好：支持遮罩批次；从张量形状获取尺寸。
- 轻量高效：计算开销极低，适合参数控制。

## 典型用法

- 驱动缩放：作为目标边长输入（如最长边）供缩放类节点使用。
- 条件分支：依据宽/高数值选择不同处理路径。

## 注意与建议

- 期望遮罩形状为 `B×H×W`；本节点仅依赖尺寸，不关心像素值具体含义。
- 批量输入时建议各帧尺寸一致，以获得确定性输出。
