# Text List to String - 文本列表合并

**节点功能：** `Text List to String` 将输入的文本列表（含嵌套列表）逐项应用前后缀后，按分隔符拼接为一个字符串。支持常见转义序列与复合分隔符，例如 `\n---\n`。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text_list` | 可选 | ANY | - | - | 文本列表或嵌套列表；缺省时输出为空字符串 |
| `prefix` | - | STRING | `""` | - | 每项前缀 |
| `suffix` | - | STRING | `""` | - | 每项后缀 |
| `separator` | - | STRING | `"\n"` | 支持 `\n` / `\t` / `\r` / `\\` 组合 | 项与项之间的连接符，支持复合分隔符如 `\n---\n` |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `text` | STRING | 合并结果字符串 |

## 功能说明

- 嵌套扁平化：自动展开子列表并按顺序参与拼接。
- 前后缀逐项应用：对每个元素分别加 `prefix` / `suffix`。
- 分隔符转义：支持 `\n`、`\t`、`\r`、`\\` 以及组合字符串（如 `\n---\n`）。
- 稳健输入：`None` 或空列表将产生空字符串输出；空白项会被保留但仍按规则处理。

## 典型用法

- 将 `List Custom String` 输出的 `string_list` 合并为段落文本（分隔符使用 `\n` 或 `\n---\n`）。
- 统一为逗号分隔的标签串：设置分隔符为 `, `，前后缀用于包装每项（如加引号）。

## 注意与建议

- 当 `text_list` 为嵌套结构时，节点会按出现顺序完整展开；如需过滤空行或注释，可先经 `String Filter` 或在上游处理。
