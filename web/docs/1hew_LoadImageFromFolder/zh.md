# Load Image From Folder - 从文件夹加载图像

**节点功能：** 从指定文件夹加载图像文件。支持批量加载、子文件夹遍历以及基于参考图像的自动缩放。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `get_image_size` | 可选 | IMAGE | - | - | 可选参考图像。如果提供，加载的图像将被缩放以匹配该图像的尺寸。 |
| `folder` | - | STRING | "" | - | 包含图像文件的文件夹绝对路径。 |
| `index` | - | INT | 0 | -8192~8192 | 要加载的图像索引（当 `all` 为 True 时忽略）。 |
| `include_subfolder` | - | BOOLEAN | True | - | 是否包含子文件夹中的图像文件。 |
| `all` | - | BOOLEAN | False | - | 如果为 True，加载文件夹中的所有图像作为一个批次。如果为 False，加载由 `index` 指定的单张图像。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 加载的图像。如果 `all` 为 True，则返回图像批次。 |
| `mask` | MASK | 对应的遮罩。如果原图包含 Alpha 通道则使用，否则生成全白遮罩。 |

## 功能说明

- **批量加载**：可以一次性加载文件夹中的所有图像（`all=True`），或按索引加载单张。
- **自动缩放**：如果连接了 `get_image_size`，所有加载的图像将被裁剪并缩放（居中裁剪+缩放）以匹配参考尺寸。
- **批次统一**：在批量模式下（`all=True`），如果未提供参考图像，将以第一张图像的尺寸为基准，后续图像都会被调整到该尺寸，以确保 Tensor 形状一致。
- **支持格式**：`.png`, `.jpg`, `.jpeg`, `.bmp`, `.webp`, `.tiff`, `.gif`。
