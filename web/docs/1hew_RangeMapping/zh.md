# Range Mapping - 范围映射（0–1 到 [min, max]）

**节点功能：** `Range Mapping` 将归一化输入 `value∈[0,1]` 映射到目标范围 `[min, max]`，并支持按位小数四舍五入或整数化输出。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `value` | - | FLOAT | `1.0` | `0.0–1.0` | 归一化输入；滑杆便于精确映射。 |
| `min` | - | FLOAT | `0.0` | 超大范围 | 映射下界。 |
| `max` | - | FLOAT | `1.0` | 超大范围 | 映射上界。 |
| `rounding` | - | INT | `3` | `0–10` | 小数位数；`0` 时按整数输出。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `float` | FLOAT | 映射后的数值（经四舍五入或整数化）。 |
| `int` | INT | 映射值的整数形式。 |

## 功能说明

- 线性映射：`actual = min + value × (max − max)`。
- 边界宽泛：`min`/`max` 支持很大的正负范围。 

## 典型用法

- 参数缩放：将滑杆值映射到 `[min, max]` 用于模型阈值或权重控制。
- 整数步进：设置 `rounding=0` 从归一化输入生成枚举索引。
- 反向映射：交换 `min` 与 `max` 以获得递减刻度。

## 注意与建议

- 建议保持 `value` 在 `0–1` 范围内；界面范围已约束。 
- 根据下游所需选择合适的 `rounding`；整数化适合离散分类或步进参数。
