# Mask Fill Hole - 遮罩填充孔洞

**节点功能：** `Mask Fill Hole`节点用于填充遮罩中的封闭区域孔洞，支持批量处理多个遮罩。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | 必选 | MASK | - | - | 需要填充孔洞的遮罩 |
| `invert_mask` | - | BOOLEAN | False | True/False | 是否反转遮罩，True为反转，False为不反转 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 填充孔洞后的遮罩 |
