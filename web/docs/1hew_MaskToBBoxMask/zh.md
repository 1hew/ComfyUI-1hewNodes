# Mask To BBox Mask - 遮罩转 BBox 遮罩

**节点功能：** `Mask To BBox Mask` 将输入遮罩的前景区域转换为最小外接矩形遮罩。适合把物体原始轮廓 mask 变成包住该物体的方形/矩形 bbox mask。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | - | MASK | - | - | 输入遮罩批次；二维输入会扩展到 `[B,H,W]` |
| `output_mode` | - | COMBO | `merge` | `merge` / `separate` | `merge` 每张输入 mask 输出一个总 bbox；`separate` 每个连通区域输出一个 bbox mask |
| `divisible_by` | - | INT | `8` | 1-1024 | 将矩形宽高向外扩展到可被该值整除；只会扩大，不会缩小原前景区域 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `bbox_mask` | MASK | 转换后的矩形 bbox 遮罩 |

## 功能说明

- 外接矩形：根据所有非零前景像素计算最小包围矩形，并把矩形区域填充为白色。
- 倍数对齐：通过 `divisible_by` 将 bbox 宽高向外补到指定倍数；靠近图像边缘时会向可用方向扩展。
- 合并输出：`merge` 模式会把同一张 mask 中所有有效前景合并为一个 bbox。
- 分离输出：`separate` 模式会按 8 连通区域分别生成 bbox mask，输出批次数量可能大于输入批次。

## 典型用法

- 物体区域框选：将一个物体的精细遮罩转换成包住它的矩形遮罩。
- 配合裁剪/粘贴：作为 `Mask Crop by BBox Mask`、`Image Paste by BBox Mask` 等节点的 bbox mask 输入。
- 多物体处理：先用 `Mask Separate` 拆出物体，再用该节点转成稳定的矩形区域。

## 注意与建议

- 当输入 mask 没有非零前景时，`merge` 模式会输出全黑 mask。
- 如果图像本身尺寸小于对齐后的目标尺寸，bbox 会扩展到图像边界内，优先保证不裁掉原前景。
- 如果一张 mask 内包含多个相隔较远的物体，`merge` 会得到包住全部物体的大矩形；需要逐个物体时请使用 `separate` 或先接 `Mask Separate`。
