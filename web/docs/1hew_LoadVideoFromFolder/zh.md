# Load Video From Folder - 从文件夹加载视频

**节点功能：** 从指定文件夹加载视频文件。支持遍历子文件夹并通过索引选择视频。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `folder` | - | STRING | "" | - | 包含视频文件的文件夹绝对路径。 |
| `index` | - | INT | 0 | -8192~8192 | 要加载的视频索引。支持负数索引（例如 -1 表示最后一个文件）。 |
| `include_subfolder` | - | BOOLEAN | True | - | 是否包含子文件夹中的视频文件。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `VIDEO` | VIDEO | 加载的视频对象。 |

## 功能说明

- **文件夹遍历**：可以从单层文件夹或递归从子文件夹加载视频。
- **索引选择**：使用整数索引选择特定视频。如果索引超出数量范围，会自动取模循环。
- **支持格式**：支持常见的视频格式，如 `.webm`, `.mp4`, `.mkv`, `.mov`, `.avi`。
