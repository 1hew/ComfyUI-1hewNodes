# Image Batch Index - 按索引文本提取图像批次

**节点功能：** `Image Batch Index` 使用多行文本索引规则从图像批次中提取指定图像。支持单个索引、列表、正序/倒序区间、区间步长，以及中英文符号和混合括号写法；超出范围的索引会自动跳过。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像批次 |
| `index` | - | STRING | `0` | 多行文本 | 索引表达式，支持中英文符号、空格、换行和混合括号 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 按索引规则提取后的图像批次 |

## 支持写法

- 单个索引：`0`、`2`、`-1`
- 列表：`0,2,6`、`0，2，6`
- 旧区间：`0-10`、`10-0`、`0-10:2`
- 区间表达式：`[0,10]`、`[0,10)`、`(0,10]`、`(0,10)`
- 混合括号：`【0，10】`、`【0，10)`、`(10，0】`
- 区间步长：`[0,20):2`、`【10，0】：2`

## 功能说明

- 宽松解析：兼容 `,`、`，`、`:`、`：`、`[]`、`【】`、`()`、`（）`，并允许混合使用。
- 多行输入：每行可以写一个或多个表达式。
- 负索引支持：如 `-1` 表示最后一张图像。
- 倒序支持：如 `10-0`、`[10,0]:2` 会按倒序展开。
- 顺序保留：按输入书写顺序输出结果。
- 重复保留：同一索引重复出现时会重复输出对应图像。
- 越界跳过：解析成功但超出批次范围的索引会被自动忽略。

## 典型用法

- 指定几张图：`0, 2, 6, -1`
- 连续取一段：`[0,10]`
- 间隔抽图：`[0,20):2`
- 倒序取图：`【10，0】：2`
- 混合输入：
  ```text
  0，2，6
  [10,20)
  【30，20】：2
  -1
  ```

## 注意与建议

- `:` / `：` 在本节点中只表示步长。
- `0-10` 默认表示两端都包含。
- 当全部索引都无效或越界时，输出为空批次。
