# Text Join Multi - 多文本连接

**节点功能：** `Text Join Multi` 节点用于将最多五个多行文本输入合并为一个字符串，支持 `{input}` 占位符替换、注释过滤、空行控制以及自定义分隔符。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text1` | 必选 | STRING | "" | 多行文本 | 第一段文本输入，按行处理 |
| `text2` | 必选 | STRING | "" | 多行文本 | 第二段文本输入 |
| `text3` | 必选 | STRING | "" | 多行文本 | 第三段文本输入 |
| `text4` | 必选 | STRING | "" | 多行文本 | 第四段文本输入 |
| `text5` | 必选 | STRING | "" | 多行文本 | 第五段文本输入 |
| `filter_empty_line` | 必选 | BOOLEAN | False | True/False | 在处理后移除空行 |
| `filter_comment` | 必选 | BOOLEAN | False | True/False | 过滤 `#` 行内注释与三引号注释块（`'''`/`"""`） |
| `separator` | 必选 | STRING | "\\n" | - | 连接分隔符；支持 \\n、\\t、\\r 转义 |
| `input` | 可选 | STRING | "" | - | 动态输入值；可在文本中以 `{input}` 引用 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `string` | STRING | 合并后的结果文本 |
