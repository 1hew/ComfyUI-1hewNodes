# Text Join Multi - 多文本拼接与过滤

**节点功能：** `Text Join Multi` 将最多五段多行文本进行拼接，支持注释与空行过滤、`{input}` 占位符替换与分隔符的转义解码。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text1` | - | STRING | `` | - | 第一段多行文本。 |
| `text2` | - | STRING | `` | - | 第二段多行文本。 |
| `text3` | - | STRING | `` | - | 第三段多行文本。 |
| `text4` | - | STRING | `` | - | 第四段多行文本。 |
| `text5` | - | STRING | `` | - | 第五段多行文本。 |
| `filter_empty_line` | - | BOOLEAN | `False` | - | 处理后移除空行。 |
| `filter_comment` | - | BOOLEAN | `False` | - | 启用注释过滤。

## 典型用法

- 组织提示词：将不同模块放入 `text1..text5`，通过 `{input}` 注入变量，并设置过滤选项以获得整洁输出。
- 语义分隔：按需选择 `separator` 为逗号、换行或制表符，以适配下游需求。

## 注意与建议

- 过滤在占位符替换之后进行，确保模板变量先被替换再执行注释剔除。
- 建议在各段文本上使用一致的过滤设置，以保持输出风格统一。
