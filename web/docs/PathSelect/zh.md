# Path Select - 路径选择

**节点功能：** `Path Select`节点提供层级结构的路径选择下拉框，支持预定义路径和自定义字段的组合。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `path` | - | COMBO[STRING] | 动态生成 | 预定义路径列表 | 层级路径选择 |
| `filename` | - | STRING | "" | 自定义文本 | 第四级自定义字段 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `filename` | STRING | 完整的路径字符串 |
