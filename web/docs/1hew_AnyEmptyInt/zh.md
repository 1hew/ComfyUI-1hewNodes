# Any Empty Int - 将“空值”映射为整数

**节点功能：** `Any Empty Int` 判定输入是否为空，并分别输出 `empty` 或 `not_empty` 的整数值。与 `Any Empty Bool` 采用相同的类型判定规则，并支持自定义映射值。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `any` | - | ANY (`*`) | - | - | 待判定的输入 |
| `empty` | - | INT | 0 | -999999–999999 | 输入为空时的输出值 |
| `not_empty` | - | INT | 1 | -999999–999999 | 输入非空时的输出值 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int` | INT | 为空输出 `empty`，非空输出 `not_empty` |

## 功能说明

- 类型判定与 `Any Empty Bool` 一致。
- 映射可配置：可按下游逻辑设置 `empty` 与 `not_empty`。
- 异常保护：发生异常时输出 `empty`。

## 典型用法

- 选择索引：用作整数选择器或循环次数的控制信号。

- 数值标记：输出 `0/1` 或自定义编码，服务于条件判断。

- 流程兜底：结合 `Any Switch Int` 或算术节点进行分支路由。

## 注意与建议

- 需要更丰富状态时，可使用更宽的取值范围（如空用 `-1`，非空用 `+1`）。
- 共享对张量/数组与容器的递归与全零判定行为。
