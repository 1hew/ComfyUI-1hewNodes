# Audio Duration - 音频时长获取

**节点功能：** `Audio Duration` 读取上游音频对象中的 `sample_rate` 与 `waveform`，输出音频的时长（秒）。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `audio` | - | AUDIO | - | - | 音频对象，包含 `sample_rate` 与 `waveform`（形如 `[batch, channels, samples]`）。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `second` | FLOAT | 音频时长（秒），计算公式 `samples / sample_rate`。 |

## 功能说明

- 稳健处理：当输入缺失或不合法时返回 `0.0`。
- 与批量无关：仅使用样本轴长度计算，声道数与批量大小不影响结果。
- 兼容单声道与双声道，无需假定样本类型。

## 典型用法

- 搭配视频保存流程，用于标注片段时长。
- 检查音频预处理结果是否满足预期时长。

## 注意与建议

- 确保上游音频提供 `sample_rate > 0` 与有效的 `waveform` 张量。
- 对于流式或分段音频，建议在合并后统一计算总时长。
