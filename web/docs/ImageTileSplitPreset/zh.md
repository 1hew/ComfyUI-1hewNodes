# Image Tile Split Preset - 图像分块分割预设

**节点功能：** `Image Tile Split Preset`节点使用预定义分辨率预设智能地将大图像分割成多个分块。提供自动尺寸选择或手动预设选择，优化分块效率并提供全面的元数据用于无缝分块处理工作流。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | 必选 | IMAGE | - | - | 要分割成分块的输入图像 |
| `overlap_amount` | 必选 | FLOAT | 0.05 | 0.0-512.0 | 重叠量：≤1.0为比例模式，>1.0为像素模式 |
| `tile_preset_size` | 必选 | COMBO[STRING] | auto | auto + 预设列表 | 分块尺寸选择：自动或预定义分辨率 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `tiles` | IMAGE | 具有一致预定义尺寸的分割图像分块批次 |
| `tiles_meta` | DICT | 全面的分块元数据，包括网格布局、效率指标和重建信息 |
