# Image BW Matte - 精简版抠图遮罩

**节点功能：** `Image BW Matte` 用固定的 `auto + soft` 流程把输入图像转成抠图遮罩，只输出 `mask`，并把可调项收敛到最常用的 3 个参数。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像，支持单图或批量 |
| `gamma` | - | FLOAT | `1.0` | `0.1 ~ 5.0` | 中间调曲线；小于 `1` 更容易保留辉光和半透明边缘 |
| `shrink_radius` | - | INT | `0` | `0 ~ 128` | 向内收边半径，单位像素；适合压掉边缘光晕或白边 |
| `blur_radius` | - | FLOAT | `0.0` | `0.0 ~ 32.0` | 最终遮罩的羽化半径 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 最终遮罩，可直接接到下游抠图、裁剪、alpha 合成节点 |

## 固定流程

- 自动来源：优先使用现成 alpha；没有有效 alpha 时，若边缘背景较稳定则走背景差分，否则走高亮值提取。
- 固定 soft：保留连续灰度，不做硬阈值二值化，更适合发光、烟雾和半透明边缘。
- 固定 levels：内部使用默认黑白场拉伸，不再暴露 `levels_low` / `levels_high`。

## 参数建议

- 普通黑底发光图：`gamma=0.9~1.0`，`shrink_radius=0~1`，`blur_radius=0~1`
- 边缘有一圈白边或光晕：先加 `shrink_radius=1~3`
- 遮罩太“硬”：保持 `shrink_radius` 小一点，再加少量 `blur_radius`
- 遮罩太“胖”：先加 `shrink_radius`，不要先猛调 `gamma`

## 关于收边参数

- 这个参数是有必要的。
- 做抠图时，主体边缘经常会残留一圈亮边、辉光或背景污染，尤其是黑底发光素材。
- `shrink_radius` 本质上是向内轻微腐蚀遮罩，通常 `1~2 px` 就够了。
- 值太大时会吃掉细节，所以建议小步调整。

## 注意与建议

- 这是一个轻量、免模型的经典遮罩节点，适合纯底、高反差、发光元素。
- 如果是复杂真实照片背景，优先考虑 `Detect Remove BG` 这类模型抠图节点。
