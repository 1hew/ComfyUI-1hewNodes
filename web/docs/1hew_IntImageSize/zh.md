# Int Image Size - 输出图像宽度与高度两个整数

**节点功能：** `Int Image Size` 直接读取输入张量的尺寸，输出图像的宽度与高度两个整数。适用于尺寸感知流程、分支判断与参数化。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像批次（`B×H×W×C`）；读取空间尺寸。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `width` | INT | 图像的宽度（像素）。 |
| `height` | INT | 图像的高度（像素）。 |

## 功能说明

- 尺寸读取：从图像张量提取 `H`、`W` 两个维度。
- 批次兼容：支持批量输入，基于张量形状使用空间尺寸。
- 轻量工具：适合用于参数控制与输入校验，开销低。

## 典型用法

- 控制缩放：将宽/高用于通用缩放或填充/裁剪逻辑。
- 条件判断：依据宽/高阈值进行流程分支或参数选择。

## 注意与建议

- 假设图像张量布局为 `B×H×W×C`；通道维度不参与计算。
- 批量输入建议各帧 `H/W` 一致，以获得确定性输出。
