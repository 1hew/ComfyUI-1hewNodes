# Image List Append - 图像列表合并

**节点功能：** `Image List Append` 将多个图像输入合并为一个列表，按输入后缀顺序（`image_1`、`image_2`、…）保序。每个输入既可为单张图像，也可为图像列表。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image_1` | 可选 | IMAGE/LIST | - | - | 第一图像或图像列表 |
| `image_2` | 可选 | IMAGE/LIST | - | - | 第二图像或图像列表 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image_list` | IMAGE_LIST | 合并后的图像列表，按输入后缀顺序排列 |

## 功能说明

- 输入灵活：支持单张或列表；`None` 输入将被忽略。
- 顺序确定：按数字后缀排序（`image_1`、`image_2`、…）。
- 列表直通：列表内容按序拼接，不复制帧。

## 典型用法

- 将不同选择器的结果合并为一个列表供下游处理。
- 通过多分支组合构建可变长度序列。

## 注意与建议

- 输出为 Python 风格的图像列表；请连接支持列表输入的节点。
