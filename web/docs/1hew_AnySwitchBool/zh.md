# Any Switch Bool - 条件选择透传

**节点功能：** `Any Switch Bool` 基于布尔值在 `on_true` 与 `on_false` 两个输入间进行选择并输出其中一个。实现惰性求值，仅计算被选中的分支。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `boolean` | - | BOOLEAN | True | - | 控制开关 |
| `on_true` | 可选 | ANY (`*`) | - | - | 当 `boolean=True` 时输出的值（惰性） |
| `on_false` | 可选 | ANY (`*`) | - | - | 当 `boolean=False` 时输出的值（惰性） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `output` | ANY (`*`) | 选中的输入值 |

## 功能说明

- 惰性端口：仅激活的输入端会被计算。
- 简单选择：`boolean=True` 输出 `on_true`，否则输出 `on_false`。
- 异常保护：发生异常时输出 `None`。

## 典型用法

- 代价分支控制：仅在条件满足时计算开销较大的分支。
- 默认路径路由：结合 `Any Empty Bool` 区分存在 vs 空值输入的分支。
- 模式切换：快速在两条处理路径间切换。

## 注意与建议

- 可提供任一或同时提供两分支输入，节点按照控制开关输出对应值。
- 结合校验节点构建安全的分支式管线。
