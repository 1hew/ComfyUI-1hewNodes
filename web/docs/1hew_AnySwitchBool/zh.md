# Any Switch Bool - 通用布尔切换

**节点功能：** `Any Switch Bool`节点是一个通用的布尔条件切换器，支持任意类型输入和惰性求值。根据布尔值条件选择输出 `on_true` 或 `on_false` 的值，只计算需要的分支，提高执行效率。适用于条件分支控制和动态数据路由。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `boolean` | 必选 | BOOLEAN | True | True/False | 控制切换的布尔条件值 |
| `on_true` | 可选 | * | - | - | 当布尔值为True时输出的值（支持任意类型） |
| `on_false` | 可选 | * | - | - | 当布尔值为False时输出的值（支持任意类型） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `output` | * | 根据布尔条件选择的输出值（类型与选中的输入一致） |

## 功能说明

### 惰性求值机制

- **智能计算**：只计算需要的分支，未选中的分支不会被执行
