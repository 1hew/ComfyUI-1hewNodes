# Text Prefix Suffix - 前后缀格式化与拼接

**节点功能：** `Text Prefix Suffix` 对输入值逐项应用 `prefix` 与 `suffix` 并按分隔符拼接。支持单值、列表/元组与一般可迭代对象的灵活输入。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `any_text` | - | ANY | - | - | 单值、列表/元组或可迭代对象；各项按字符串化处理。 |
| `prefix` | - | STRING | `` | - | 每项前置文本。 |
| `suffix` | - | STRING | `` | - | 每项后置文本。 |
| `separator` | - | STRING | `\\n` | - | 拼接分隔符；当取值为 `\\n` 时输出换行，其余按原样使用。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `text` | STRING | 按每项格式化后拼接得到的文本。 |

## 功能说明

- 可迭代处理：对非字符串的可迭代对象展开为列表，否则包装为单项列表` 使用。

## 典型用法

- 列表格式化：输入列表或生成器，设置前后缀以获得统一包裹。
- 多行输出：保持 `separator=\\n` 实现每项一行的版式。
- 自定义分隔：设置 `separator=", "` 或其他符号以生成紧凑字符串。

## 注意与建议

- 如需对 `\t`、`\r` 或 `\\` 等转义进行解码，可与 `Text Join by Text List` 或 `Text Join Multi` 配合。
- 支持一般可迭代对象，便于直接格式化生成器或集合。
