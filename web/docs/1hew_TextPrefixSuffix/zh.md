# Text Prefix Suffix - 文本前后缀

**节点功能：** `Text Prefix Suffix` 对输入文本（或文本列表）逐项添加前缀与后缀，并以换行 `\n` 拼接为单个字符串输出。适合为每行统一加标签、括号、标记等格式化场景。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text` | 可选 | ANY | - | - | 任意文本或可迭代对象；可迭代且非字符串时会转换为列表进行逐项处理 |
| `prefix` | - | STRING | `""` | - | 每项前缀（如 `[`、`Scene: `） |
| `suffix` | - | STRING | `""` | - | 每项后缀（如 `]`、`;`） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `text` | STRING | 逐项应用前后缀并以换行拼接后的字符串 |

## 功能说明

- 逐项装饰：对输入中的每个元素分别应用 `prefix` 和 `suffix`。
- 自动归一：
  - 若 `text` 不是列表或元组，但为非字符串的可迭代对象（如生成器），将转换为列表后再处理；
  - 若 `text` 为单个字符串或标量，则封装为单元素列表后处理。
- 固定分隔：输出使用换行符 `\n` 进行拼接，保持各项独立成行。

## 典型用法

- 为每行加括号：`prefix="["`、`suffix="]"`。
- 为多段场景说明加统一标签：`prefix="Scene: "`、`suffix=""`。

## 注意与建议

- 如需自定义分隔符（例如 `\n---\n` 或其他复合分隔符），请使用 `Text List to String` 或 `String Join Multi` 节点，它们支持分隔符转义与复合形式。
- 输入包含嵌套列表时，内部列表将按 Python 的 `str(item)` 结果参与拼接；若需扁平化嵌套结构，请在上游进行列表展开或改用支持扁平化的节点。
