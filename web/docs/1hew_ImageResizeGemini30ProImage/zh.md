# Image Resize Gemini30ProImage - Gemini 3.0 Pro 图像尺寸适配

**节点功能：** `Image Resize Gemini30ProImage` 用于将输入图像/遮罩适配到 Gemini 3.0 Pro 常用预设分辨率，支持自动匹配最接近分辨率，并在 `crop` / `pad` / `stretch` 三种模式下同步输出图像与遮罩。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `preset_size` | - | COMBO | `auto (2k \| 4k)` | `auto` / `auto (1k \| 2k)` / `auto (2k \| 4k)` / 各预设分辨率项 | 目标尺寸选择；`auto*` 会按输入宽高比与面积自动匹配最接近预设 |
| `fit` | - | COMBO | `crop` | `crop` / `pad` / `stretch` | 适应模式：裁剪、填充、拉伸 |
| `pad_color` | - | STRING | `1.0` | 灰度/HEX/RGB/颜色名/`edge`/`average`/`extend`/`mirror` | `pad` 模式背景填充策略 |
| `image` | 可选 | IMAGE | - | - | 输入图像批次 |
| `mask` | 可选 | MASK | - | - | 输入遮罩批次，可与图像同步变换 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 调整后的图像批次 |
| `mask` | MASK | 与输出尺寸对齐的遮罩批次 |

## 功能说明

- 自动分辨率匹配：在预设列表中按宽高比优先、面积次优进行最近邻匹配。
- 支持无图像或无遮罩场景：只输入 `image` 或 `mask` 也可执行；都为空时输出目标尺寸背景图与全白遮罩。
- 三种适配模式：
  - `crop`：先按比例中心裁剪再缩放到目标尺寸。
  - `pad`：先按比例缩放再居中填充到目标尺寸。
  - `stretch`：直接拉伸到目标尺寸。
- 遮罩同步：存在输入遮罩时，随图像使用同策略变换并输出。

## 典型用法

- 适配 Gemini 3.0 Pro 常见输入尺寸：`preset_size=auto (2k | 4k)`，`fit=crop`。
- 保留完整画面并补边：`fit=pad`，按需要设置 `pad_color`（如 `1.0`、`#000000`、`mirror`）。
- 只用遮罩对齐目标分辨率：只连接 `mask` 输入，节点会生成对应背景图并输出对齐后的遮罩。

## 注意与建议

- `pad_color` 在 `pad` 模式下生效最明显；`crop`/`stretch` 主要影响几何变换。
- 当 `image` 与 `mask` 尺寸不一致时，建议先统一尺寸再送入，以获得更稳定的边缘对齐效果。
