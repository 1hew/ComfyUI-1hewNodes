# Image to Mask - 将图像亮度转换为遮罩

**节点功能：** `Image to Mask` 将可选的图像输入按 RGB 亮度转换为灰度 `MASK`。当未连接图像时，输出 64x64 的全黑遮罩。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | 可选 | - | 要转换为遮罩的图像；未连接时输出 64x64 全黑遮罩 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 从图像亮度生成的灰度遮罩；未连接图像时为 64x64 全黑遮罩 |

## 功能说明

- 图像输入可选，适合需要 fallback mask 的工作流。
- 使用 RGB 亮度权重：`0.299 * R + 0.587 * G + 0.114 * B`。
- 连接图像时保留原始 batch 数量和图像尺寸。
- 未连接图像时输出 batch=1 的 64x64 全黑遮罩。

## 典型用法

- 将灰度图或 RGB 引导图转换为 mask。
- 为可选图像分支提供安全的黑色遮罩兜底。
- 为下游遮罩操作节点准备基于图像亮度的遮罩。
