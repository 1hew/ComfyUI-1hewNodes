# Coordinate Extract - 坐标提取器

**节点功能：** `Coordinate Extract`节点用于从JSON格式的坐标数据中提取X和Y坐标列表，支持批量坐标处理。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `coordinates_json` | - | STRING | - | 多行文本 | JSON格式的坐标数据 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `x` | FLOAT | X坐标列表 |
| `y` | FLOAT | Y坐标列表 |

## 功能说明

### 输入示例
```json
[
    {
        "x": 0,
        "y": 512
    },
    {
        "x": 59,
        "y": 510
    }
]
