# List Custom Float - 宽松语法浮点列表

**节点功能：** `List Custom Float` 将多行文本解析为浮点数列表，并同时输出列表及其长度。支持单值、列表、正序/倒序区间、区间步长，以及中英文符号、空格、换行和混合括号写法；浮点区间会做精度清理，避免常见的小数累积误差。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `custom_text` | - | STRING | `` | 多行文本 | 待解析的多行文本，支持宽松区间语法。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `float_list` | LIST | 解析得到的浮点数列表（列表型输出端）。 |
| `count` | INT | `float_list` 的长度。 |

## 功能说明

- 宽松解析：兼容 `,`、`，`、`:`、`：`、`[]`、`【】`、`()`、`（）`，并允许混合使用。
- 单值与列表：支持 `1.0`、`.5`、`1.0, 2.5, 3`。
- 区间与步长：支持 `0-1`、`1-0`、`0-1:0.25`、`[0,1]`、`[0,1)`、`【1，0】：0.2`。
- 倒序支持：如 `1-0`、`[1,0]:0.2` 会按倒序展开。
- 精度清理：输出前统一清理浮点噪声，避免如 `0.30000000000000004`。
- 顺序与重复保留：输出顺序按书写顺序，重复项会保留。
- 稳健缺省：空输入返回 `[0.0]`；无法解析的片段会被忽略。

## 典型用法

- 单值与列表：`1.0, 0.5, 2.75`
- 连续区间：`[0,1]`
- 间隔区间：`[0,1):0.25`
- 倒序区间：`【1，0】：0.2`
- 混合输入：
  ```text
  1.0，0.5，2.75
  [0,1):0.25
  【1，0】：0.2
  .125
  ```

## 注意与建议

- `:` / `：` 在本节点中只表示步长。
- 区间端点和步长都支持浮点数。
- 当不存在有效数值时将回退为 `[0.0]`。
