# List Custom Float - 自定义浮点列表

**节点功能：** `List Custom Float` 将多行文本解析为浮点数列表，并同时输出列表及其长度。支持破折线分段模式与多分隔符的 CSV 风格解析，并处理项级引号。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `custom_text` | - | STRING | `` | - | 待解析的多行文本。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `float_list` | LIST | 解析得到的浮点数列表（列表型输出端）。 |
| `count` | INT | `float_list` 的长度。 |

## 功能说明

- 破折线分段：仅由 `-` 构成的行划分段落；每段会去除包裹引号并按浮点数解析。
- CSV 解析：支持 `,`、`;`、`，`、`；` 等分隔符；逐项去除引号。
- 稳健缺省：空输入返回 `[0.0]`；无法解析的项会被忽略。
- 核心逻辑：破折线检测；分段解析；CSV 解析。

## 典型用法

- CSV 输入：将 `1.0, 0.5, 2.75` 写在一行或多行。
- 分段模式：用由破折线组成的分隔行表示独立数值段。

## 注意与建议

- 数值项外层引号（如 `'0.5'`）会在解析前去除。
- 当不存在有效数值时将回退为 `[0.0]`。
