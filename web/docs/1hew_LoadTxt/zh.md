# Load Txt - 加载文本文件

**节点功能：** `Load Txt` 用于从文本文件路径或目录中读取 `.txt` 内容，并输出文本内容与文件名。支持目录扫描、递归子目录、`encode` 选择，以及通过 `index` 在多个文本文件中按序取值。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `file` | - | STRING | `""` | - | 文本文件路径或目录路径。支持绝对路径，也会尝试在 ComfyUI 的 input/output/temp 目录中解析相对路径。 |
| `encode` | - | COMBO | `auto` | `auto` / `utf-8` / `utf-8-sig` / `gbk` / `utf-16` | 文本读取编码；`auto` 会按常见编码顺序自动尝试。 |
| `index` | - | INT | `0` | -8192-8192 | `file` 为目录时的文本索引；支持负数索引（按数量取模）。 |
| `include_subdir` | - | BOOLEAN | `false` | - | `file` 为目录时是否包含子目录。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `text` | STRING | 读取到的文本内容。 |
| `filename` | STRING | 当前选中文本文件名（不含扩展名）。 |

## 功能说明

- 文件/目录输入：支持直接读取单个 `.txt` 文件，也支持从目录内选择文本文件。
- 编码兼容：`auto` 模式会依次尝试 `utf-8`、`utf-8-sig`、`gbk`、`utf-16`，尽量减少乱码。
- 稳定排序：目录模式下按大小写无关排序，保证 `index` 行为更稳定。
- 索引选择：目录模式下用 `index` 选择目标文件，负数索引按数量取模。
- 界面预览：在 UI 面板直接显示读取到的文本内容。

## 典型用法

- 从提示词目录中按索引读取不同 `.txt` 模板，驱动批量工作流。
- 读取单个配置文本文件，并把 `text` 输出连接到文本处理或提示词节点。

## 注意与建议

- 当前仅匹配 `.txt` 扩展名文件。
- 当目录中没有可用文本文件，或路径不存在时，节点会报错。
