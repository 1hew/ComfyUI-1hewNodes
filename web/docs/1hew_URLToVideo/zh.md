# URL to Video - 从 URL 下载视频

**节点功能：** `URL to Video` 从 HTTP(S) 链接下载视频并输出视频对象，便于下游处理。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `video_url` | - | STRING | - | http(s) | 视频资源的直链，必须以 `http://` 或 `https://` 开头 |
| `timeout` | - | INT | 30 | 5-300 | 下载超时时间（秒） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `video` | VIDEO | 根据下载字节构建的视频对象 |

## 功能说明

- 流式下载：以桌面浏览器 UA 进行分块拉取，提升兼容性。
- 错误处理：对无效 URL 或下载失败进行日志记录并抛出异常。
- 兼容性：返回 `VideoFromFile` 对象，适配下游节点。

## 典型用法

- 提供视频文件的直链进行下载并转为视频对象。

## 注意与建议

- 请确保链接指向可下载的视频内容（非 HTML 页面或受 DRM 保护的流）。
