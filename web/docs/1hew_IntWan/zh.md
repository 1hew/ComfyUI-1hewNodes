# Int Wan - 整数常量

**节点功能：** `Int Wan` 输出一个整数常量，在执行时将输入转换为 `int`，并遵循范围与界面步进设置。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `value` | - | INT | `1` | `1–10000` | 输出的整数值。界面步进为 `4`，便于快速调节。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int` | INT | 透传的整数常量。 |

## 功能说明

- 类型安全：在执行阶段对输入做 `int` 转换。
- 常量供给：适合在图中作为下游节点参数来源。
- 核心逻辑：内部类型转换与常量输出。

## 典型用法

- 为后续节点提供稳定的整数常量（如计数、索引）。

## 注意与建议

- 当无需细粒度调节时可借助步进值 `4` 快速移动。
