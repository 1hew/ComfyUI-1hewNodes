# Text Join by Text List - 列表项拼接

**节点功能：** `Text Join by Text List` 将列表中的各项按顺序拼接为单一文本，并对每项应用 `prefix` 与 `suffix`。分隔符支持常见的转义序列。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text_list` | - | ANY | - | - | 输入为列表或单值；单值会包装为单项列表。 |
| `prefix` | - | STRING | `` | - | 每项前置文本。 |
| `suffix` | - | STRING | `` | - | 每项后置文本。 |
| `separator` | - | STRING | `\\n` | - | 拼接分隔符；支持 `\\n`、`\\t`、`\\r`、`\\\\`。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `text` | STRING | 按格式化后各项拼接得到的文本。 |

## 功能说明

- 分隔符转义：解码 `\\n`→`\n`、`\\t`→`\t`、`\\r`→`\r`、`\\\\`→`\\`` 转换，数值或混合类型均可顺畅拼接。
