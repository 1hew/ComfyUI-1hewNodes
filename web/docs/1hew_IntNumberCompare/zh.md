# Int Number Compare - 整数数值比较

**节点功能：** `Int Number Compare` 比较两个整数输入，并根据选择的比较符输出布尔值。适用于基于整数阈值、计数或索引的条件判断。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `a` | - | INT | 0 | -999999999–999999999 | 左侧整数值 |
| `operator` | - | COMBO | `==` | `==`、`!=`、`>`、`>=`、`<`、`<=` | 比较运算符 |
| `b` | - | INT | 0 | -999999999–999999999 | 右侧整数值 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `bool` | BOOLEAN | 比较结果，条件成立为 `true`，否则为 `false` |

## 功能说明

- 支持常用整数比较：等于、不等于、大于、大于等于、小于、小于等于。
- 输入会按整数处理，适合连接计数、索引、开关选择值等整数结果。
- 异常保护：发生异常时输出 `false`。

## 典型用法

- 阈值判断：判断计数是否达到指定数量。
- 分支控制：将输出连接到 `Any Switch Bool` 进行流程选择。
- 索引判断：根据当前索引是否等于某个目标值触发特定逻辑。
