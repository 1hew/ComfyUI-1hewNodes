# List Custom Seed - 自定义种子列表

**节点功能：** `List Custom Seed` 基于输入 `seed` 生成一组唯一的随机种子，并同时输出列表及其长度。所有种子都会被钳制到合法范围。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `seed` | - | INT | `42` | `0–1125899906842624` | 初始随机种子；用于为 Python RNG 设定可复现的状态。 |
| `count` | - | INT | `3` | `1–1000` | 生成的唯一种子数量。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `seed_list` | LIST | 生成的唯一种子列表（列表型输出端）。 |
| `count` | INT | `seed_list` 的长度。 |

## 功能说明

- 可复现生成：以输入 `seed` 对 RNG 进行播种。
- 单次调用内唯一：通过内部集合与有界重试策略避免重复。
- 范围安全：每个种子被钳制到 `0..1125899906842624`。
- 核心逻辑：执行；唯一生成；钳制。

## 典型用法

- 批量生成：设置 `seed=12345`、`count=8` 以获得 8 个可复现的种子。
- 批处理图：使用列表型输出驱动每项的随机性。 

## 注意与建议

- 节点通过最多 `count×10` 次尝试与回退机制确保所需数量的唯一种子。
- 每次调用相互独立；提供确定性的 `seed` 可复现输出。
