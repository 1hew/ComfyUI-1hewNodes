# Float Number Compare - 浮点数值比较

**节点功能：** `Float Number Compare` 比较两个浮点数输入，并根据选择的比较符输出布尔值。适用于基于小数参数、比例或阈值的条件判断。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `a` | - | FLOAT | 0.0 | -999999999.0–999999999.0 | 左侧浮点数值 |
| `operator` | - | COMBO | `==` | `==`、`!=`、`>`、`>=`、`<`、`<=` | 比较运算符 |
| `b` | - | FLOAT | 0.0 | -999999999.0–999999999.0 | 右侧浮点数值 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `bool` | BOOLEAN | 比较结果，条件成立为 `true`，否则为 `false` |

## 功能说明

- 支持常用浮点比较：等于、不等于、大于、大于等于、小于、小于等于。
- 输入会按浮点数处理，适合连接比例、透明度、缩放值、阈值等小数参数。
- 异常保护：发生异常时输出 `false`。

## 典型用法

- 阈值判断：判断某个小数参数是否超过指定阈值。
- 分支控制：将输出连接到 `Any Switch Bool` 进行流程选择。
- 参数验证：检查比例、透明度或强度类参数是否处于目标范围。
