# Int Image Count - 统计有效图像数量

**节点功能：** `Int Image Count` 从动态 `image_1..image_N` 输入中统计有效图像数量，空输入、空张量与全黑/全 0 图像不会计入。适合把动态参考图数量转换为整数参数。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image_1` | - | IMAGE | - | - | 首个图像输入；前端会按连接情况动态追加 `image_2..image_N`。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int` | INT | 有效图像数量。 |

## 功能说明

- 动态输入：连接最后一个 `image_X` 后会自动追加下一个图像端口。
- 有效性判断：未连接、空张量、全 0 / 全黑图像计为无效。
- 批次兼容：当某个输入是批量图像时，会逐张检查并计数有效帧。

## 典型用法

- 参考图数量：将多个可选参考图的实际有效数量输出给后续节点。
- 动态参数：配合支持整数控制的节点，根据有效图像数量调整流程。

## 注意与建议

- 判断逻辑与 `Gemini 3.1 Flash Image Preview` 的参考图过滤语义保持一致。
- 如果需要把纯黑图作为有效内容，请先对图像做微小非零处理，或使用其他显式计数方式。
