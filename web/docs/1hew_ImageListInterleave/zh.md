# Image List Interleave - 图像列表分段交错重排

**节点功能：** `Image List Interleave` 将输入图像列表按 `segment_count` 拆成若干连续分段，再按“每段第 1 张、每段第 2 张……”的列优先顺序重新输出。它只改变列表顺序，不会统一尺寸，因此适合不同尺寸图片的重排与保存。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE / IMAGE_LIST | - | - | 输入图像；可接批次或列表，批次会拆成单张列表项 |
| `segment_count` | - | INT | `2` | `1-100000` | 将输入列表分成多少段，再按列交错重排 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image_list` | IMAGE_LIST | 重排后的图像列表 |

## 功能说明

- 连续分段：按原顺序把列表切成 `segment_count` 个连续小段。
- 列优先交错：先输出每段第 1 张，再输出每段第 2 张，以此类推。
- 尺寸保持：不做 padding、crop 或 resize，每张图保持原始尺寸。
- 输入兼容：批次输入会拆为单张 `[1,H,W,C]`，列表输入会递归展开。

## 典型示例

- 输入 `1, 2, 3, 4, 5, 6`，`segment_count=3`
  - 分段后：`(1, 2)`、`(3, 4)`、`(5, 6)`
  - 输出后：`1, 3, 5, 2, 4, 6`

- 输入 `1, 2, 3, 4, 5, 6, 7`，`segment_count=3`
  - 分段后：`(1, 2, 3)`、`(4, 5)`、`(6, 7)`
  - 输出后：`1, 4, 6, 2, 5, 7, 3`

## 注意与建议

- 当 `segment_count=1` 时，输出顺序与输入一致。
- 当 `segment_count` 大于输入数量时，实际会按输入数量处理，每段最多 1 张图，因此输出也会保持原顺序。
- 需要保持不同图片尺寸并连接 `Save Image` 时，优先使用该节点而不是 batch 版本。
