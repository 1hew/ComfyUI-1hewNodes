# Mask Batch to List - 遮罩批次拆分为单帧列表

**节点功能：** `Mask Batch to List` 将多帧遮罩批次转换为 Python 列表，其中每个元素为单帧批次（`B=1`），保持顺序与张量属性。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask_batch` | - | MASK | - | - | 输入遮罩批次（`B×H×W`） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask_list` | MASK_LIST | 单帧遮罩批次列表，顺序与输入一致 |

## 功能说明

- 确定性拆分：逐帧切片并保持 `B=1` 批次形态。
- 空输入处理：当输入为空或 `B=0` 时返回空列表。

## 典型用法

- 将遮罩序列转换为列表供逐帧处理节点使用。

## 注意与建议

- 列表元素形状为批次（`1×H×W`）。
