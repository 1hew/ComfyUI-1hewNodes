# List Custom Int - 自定义整数列表

**节点功能：** `List Custom Int` 将多行文本解析为整数列表，并同时输出列表及其长度。支持破折线分段与多分隔符的 CSV 风格解析，并处理项级引号与小数到整数的转换。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `custom_text` | - | STRING | `` | - | 待解析的多行文本。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int_list` | LIST | 解析得到的整数列表（列表型输出端）。 |
| `count` | INT | `int_list` 的长度。 |

## 功能说明

- 破折线分段：每个由破折线组成的段表示一个整数项。
- CSV 解析：支持 `,`、`;`、`，`、`；` 等分隔符；逐项去除引号。
- 小数整化：如 `"3.9"` 会按 `int(float(...))` 转为 `3`。
- 稳健缺省：空输入返回 `[0]`；无法解析的项会被忽略。
- 核心逻辑：破折线检测；分段解析；CSV 解析；小数整化。

## 典型用法

- CSV 输入：将 `1, 2, 3` 写在一行或多行。
- 混合数值字符串：如 `"7"`、`"8.2"`、`"9"` 解析为 `7, 8, 9`。
- 分段模式：用破折线分隔行表示独立整数项。

## 注意与建议

- 数值项外层引号在解析前会去除。
- 当不存在有效整数时将回退为 `[0]`。
