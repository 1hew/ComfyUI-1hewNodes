# List Custom Int - 宽松语法整数列表

**节点功能：** `List Custom Int` 将多行文本解析为整数列表，并同时输出列表及其长度。支持单值、列表、正序/倒序区间、区间步长，以及中英文符号、空格、换行和混合括号写法；单个浮点值会按 `int(float(...))` 转成整数。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `custom_text` | - | STRING | `` | 多行文本 | 待解析的多行文本，支持宽松区间语法。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int_list` | LIST | 解析得到的整数列表（列表型输出端）。 |
| `count` | INT | `int_list` 的长度。 |

## 功能说明

- 宽松解析：兼容 `,`、`，`、`:`、`：`、`[]`、`【】`、`()`、`（）`，并允许混合使用。
- 单值与列表：支持 `1`、`2.9`、`1,2,3`、`1，2，3`。
- 区间与步长：支持 `0-10`、`10-0`、`0-10:2`、`[0,10]`、`[0,10)`、`【10，0】：2`。
- 倒序支持：如 `10-0`、`[10,0]:2` 会按倒序展开。
- 顺序与重复保留：输出顺序按书写顺序，重复项会保留。
- 稳健缺省：空输入返回 `[0]`；无法解析的片段会被忽略。

## 典型用法

- 单值与列表：`1, 2, 3`
- 连续区间：`[0,10]`
- 间隔区间：`[0,20):2`
- 倒序区间：`【10，0】：2`
- 混合输入：
  ```text
  1，2，3
  [10,20)
  【30，20】：2
  7.9
  ```

## 注意与建议

- `:` / `：` 在本节点中只表示步长。
- 单个浮点值会按 `int(float(...))` 转换，例如 `7.9 -> 7`。
- 区间端点和步长按整数语义解析。
- 当不存在有效整数时将回退为 `[0]`。
