# Multi String Join - 按序拼接、过滤与占位符替换

**节点功能：** `Multi String Join` 按数字顺序拼接多个字符串，支持自定义分隔符、过滤空行与注释，并进行 `{input}` 占位符替换。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `filter_empty_line` | - | BOOLEAN | False | - | 处理后移除空行 |
| `filter_comment` | - | BOOLEAN | False | - | 移除注释与三引号块 |
| `separator` | - | STRING | `\n` | - | 拼接分隔符；支持转义 `\n`、`\t`、`\r` |
| `input` | - | STRING | `` | - | 用于替换 `{input}` 占位符的值 |
| `string_1` | - | STRING | - | - | 第一段字符串 |
| `string_2…string_N` | 可选 | STRING | - | - | 其他字符串，按数字后缀排序识别 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `string` | STRING | 过滤并拼接后的最终字符串 |

## 功能说明

- 顺序采集：按 `string_*` 的数字后缀收集并保序。
- 占位符替换：在各段与最终结果中替换 `{input}`。
- 注释过滤：移除 `#` 注释，并以状态机方式处理 `'''`/`"""` 的块注释。
- 空行过滤：可选在修剪后移除空行。
- 分隔符解码：支持转义序列以更灵活的拼接。

## 典型用法

- 提示词拼装：组合多段提示词，并用占位符注入动态内容。
- 配置模板：按段拼接并过滤注释与空行，得到干净文本。
- 顺序合并：用 `string_1..N` 保证拼接顺序符合预期。

## 注意与建议

- 三引号块处理保证块内多行内容在过滤注释时被保留。
- 可使用 `separator="\n\n"` 获得段落间隔，或 `", "` 构建逗号分隔列表。
