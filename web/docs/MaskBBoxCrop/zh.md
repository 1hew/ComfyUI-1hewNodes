# Mask BBox Crop - 蒙版边界框裁剪

**节点功能：** `Mask BBox Crop`节点根据边界框信息批量裁剪蒙版，支持多图像处理和精确的区域提取功能。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | 必选 | MASK | - | - | 要裁剪的输入蒙版 |
| `bbox_meta` | 必选 | DICT | - | - | 边界框元数据，包含裁剪位置和尺寸信息 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `cropped_mask` | MASK | 裁剪后的蒙版 |
