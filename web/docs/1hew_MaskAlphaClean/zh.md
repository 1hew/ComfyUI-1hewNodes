# Mask Alpha Clean - 清理遮罩 Alpha 噪点

**节点功能：** `Mask Alpha Clean` 用于清理遮罩中的弱透明噪点和细小孤立连通域，采用简洁强度预设，适合在常规流程中快速使用。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | - | MASK | - | - | 输入遮罩批次；二维遮罩会自动扩展为 `[B,H,W]` |
| `clean_strength` | - | COMBO | `balanced` | `soft / balanced / strong` | 清理强度预设，控制阈值与小连通域去除力度 |
| `detect_only` | - | BOOLEAN | False | - | 为 `True` 时仅检测噪点区域，输出遮罩保持不变 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 清理后的遮罩批次（`detect_only=True` 时为原始遮罩） |
| `noise_mask` | MASK | 检测出的噪点区域（`1` 表示疑似噪点） |

## 功能说明

- 预设模式：`soft / balanced / strong`，减少手动调参成本。
- 噪点去除：清理弱透明碎点与小面积孤立区域。
- 检测模式：可先定位问题区域再执行正式清理。

## 典型用法

- 分割后清理遮罩，提升后续裁剪/回贴质量。
- 去除抠图细化流程中产生的零散漂浮点。
- 先用 `detect_only` 检查清理影响，再导出最终遮罩。

## 注意与建议

- 细线条较多的遮罩建议优先使用 `soft`。
- 杂点较多时可切换 `strong` 获得更强清理效果。
