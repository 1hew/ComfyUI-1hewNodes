# Int Wan - 整数万

**节点功能：** `Int Wan`节点生成遵循4n+1序列模式的整数（1, 5, 9, 13, 17...），为需要特定等差数列的工作流提供专用整数输入。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `value` | 必选 | INT | 1 | 1-10000 | 遵循4n+1序列模式的整数值，步长为4 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `value` | INT | 遵循4n+1序列的输入整数值 |
