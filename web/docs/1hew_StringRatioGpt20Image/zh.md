# String Ratio Gpt20Image - GPT 2.0 图像比例选择器

**节点功能：** `String Ratio Gpt20Image` 为每张输入图像输出最接近的 GPT 2.0 图像模型支持比例字符串；如果未连接图像，则直接输出手动选择的比例。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `selection` | - | COMBO | `3:2` | `3:2` / `1:1` / `2:3` / `5:4` / `4:5` / `16:9` / `9:16` / `21:9` / `3:4` / `4:3` | 手动选择的 GPT 2.0 图像比例预设；未连接 `image` 时直接输出此值。 |
| `image` | 可选 | IMAGE | - | - | 输入图像批次，用于逐张推断最接近的支持比例。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `string` | STRING | 比例字符串结果；批量输入时按批次顺序以换行拼接输出。 |

## 功能说明

- GPT 2.0 图像比例集：使用你提供截图中的比例集合。
- 图像驱动推断：基于对数比例差匹配最接近的预设宽高比。
- 手动回退：未连接图像时直接透传手动选择的比例。
- 批处理友好：支持图像批次输入，每张图输出一行比例字符串。

## 典型用法

- 用在需要比例字符串输入的 GPT 2.0 图像节点前。
- 如果希望下游固定使用某个比例，可以不连接 `image`，直接手动选择目标值。

## 注意与建议

- 这个节点只输出比例字符串，不会修改图像尺寸。
- 该节点只包含 GPT 2.0 图像模型这组比例，不包含 Gemini 节点中的极端比例预设。
