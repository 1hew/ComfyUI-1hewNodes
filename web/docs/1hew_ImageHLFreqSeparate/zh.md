# Image HL Freq Separate - 高频/低频分离

**节点功能：** `Image HL Freq Separate` 以 `rgb`、`hsv` 或 `igbi` 方法将图像分离为高频与低频成分，并返回分离层与按所选方法重组的预览图。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像批次 |
| `method` | - | COMBO | `rgb` | `rgb` / `hsv` / `igbi` | 分离方法 |
| `blur_radius` | - | FLOAT | 10.0 | 0.0–100.0 | 高斯模糊半径；内部确保奇数且 ≥3 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `high_freq` | IMAGE | 高频层批次 |
| `low_freq` | IMAGE | 低频层批次 |
| `combine` | IMAGE | 使用所选方法的重组结果预览 |

## 功能说明

- 奇数半径：模糊半径会被调整为奇数且不小于 3。

## 典型用法

- 细节修图：提取高频处理纹理、低频处理色调；用 `combine` 预览预期重组效果。
- 频域融合：将 `high_freq`、`low_freq` 输入 `Image HL Freq Combine` 以进行强度可控的重组。

## 注意与建议

- 输入转换为 NumPy 以进行 OpenCV 操作；输出为 `[0,1]` 浮点图。
- `combine` 输出与分离方法一致，用于直观展示重组效果。
