# Multi Image Batch - 多图像批次

**节点功能：** `Multi Image Batch` 节点从动态 `image_X` 输入构建图像批次，以上一端口图像尺寸为参照，按 `pad`、`crop`、`stretch` 统一尺寸，支持边缘取色填充与顺序一致的收集。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image_1` | 可选 | IMAGE | - | - | 动态图像输入首端口；可扩展 `image_2`、`image_3` 等 |
| `fit` | 必选 | COMBO[STRING] | pad | crop, pad, stretch | 尺寸统一模式 |
| `pad_color` | 必选 | STRING | 1.0 | 颜色字符串 | 填充颜色；支持多种格式，`edge`/`e` 使用对应边缘平均色 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `images` | IMAGE | 统一后的图像批次，数值范围钳制为 `[0, 1]` |
