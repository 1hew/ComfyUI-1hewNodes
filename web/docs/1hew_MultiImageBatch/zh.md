# Multi Image Batch - 多图批次尺寸对齐

**节点功能：** `Multi Image Batch` 将多张图像按第一张的尺寸进行对齐，支持 `crop`、`pad`、`stretch` 三种适配模式与高级补边颜色策略，最终将对齐后的图像拼接为一个批次。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `fit` | - | COMBO | `pad` | `crop` / `pad` / `stretch` | 对齐模式 |
| `pad_color` | - | STRING | `1.0` | 颜色名/HEX/RGB/`edge`/`average`/`extend`/`mirror` | 补边颜色或策略 |
| `image_1` | - | IMAGE | - | - | 第一张图像；定义参考尺寸 |
| `image_2…image_N` | 可选 | IMAGE | - | - | 额外图像，按数字后缀排序识别 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 对齐后的图像批次；裁剪到 `[0,1]` 浮点 |

## 功能说明

- 参考尺寸：使用 `image_1` 的高宽作为目标。
- 适配模式：
- `stretch`：直接双三次缩放到目标。
- `crop`：缩放到覆盖目标后居中裁切。
- `pad`：缩放到适配目标内后按 `pad_color` 居中补边。
- 高级补边：`extend` 复制边缘、`mirror` 分段反射、`edge` 边缘均值、`average` 全局均值，或显式颜色。
- 顺序采集：按 `image_*` 的数字后缀收集并保序。

## 典型用法

- 批次统一：在入模或堆叠前统一所有图像尺寸。
- 安全补边：选择 `extend/mirror` 获得自然边界，`edge` 协调色彩，`average` 获取统一色调。
- 覆盖与保全：`crop` 保证覆盖内容，`pad` 保留完整图像内容。

## 注意与建议

- 第一张图像决定目标尺寸；确保其符合预期输出大小。
- 颜色解析支持灰度浮点、`R,G,B`（`0..1` 或 `0..255`）、`#hex` 与常见颜色名。
