# Mask List to Batch - 列表合并为填充遮罩批次

**节点功能：** `Mask List to Batch` 将遮罩列表合并为单个批次。统一形状并按最大高宽进行零值填充，以便安全拼接。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask_list` | 列表 | MASK/LIST | - | - | 遮罩列表或单张遮罩 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask_batch` | MASK | 经过零值填充到公共尺寸后的遮罩批次（`B×H×W`） |

## 功能说明

- 输入灵活：接受单张或列表/元组；非张量元素自动忽略。
- 形状统一：将 `H×W` 转为 `1×H×W` 以便拼接。
- 尺寸对齐：对每张遮罩按所有输入的最大高宽进行零值填充。
- 回退策略：当无有效遮罩时返回空批次 `(0, 64, 64)`；仅 1 张遮罩时直接返回该遮罩。

## 典型用法

- 将不同尺寸的遮罩合并为批次以对齐模型输入。

## 注意与建议

- 填充采用常数 `0`，并在拼接前保持设备与数据类型一致。
