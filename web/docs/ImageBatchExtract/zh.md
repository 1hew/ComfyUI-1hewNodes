# Image Batch Extract - 批量图像提取

**节点功能：** `Image Batch Extract`节点用于从批量图像中提取特定的图像，支持多种提取模式：自定义索引、步长间隔、总帧数自动计算，常用于视频帧提取和图像序列处理。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | 必选 | IMAGE | - | - | 输入的批量图像 |
| `mode` | - | COMBO[STRING] | step | index, step, uniform | 提取模式：index(自定义索引)、step(步长间隔)、uniform(均匀分布) |
| `index` | - | STRING | "0" | 自定义索引 | 自定义索引字符串，支持逗号分隔的索引列表，如"0,2,5,10" |
| `step` | - | INT | 4 | 1-8192 | 步长间隔，每隔多少帧提取一张图像 |
| `uniform` | - | INT | 4 | 0-8192 | 均匀分布数量，从总帧数中均匀提取指定数量的图像 |
| `max_keep` | - | INT | 10 | 0-8192 | 最大保留数量，限制最终输出的图像数量 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 提取后的图像批次 |
