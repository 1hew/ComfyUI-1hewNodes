# Image Batch Extract - 批量图像提取

**节点功能：** `Image Batch Extract`节点用于从批量图像中提取特定图像，支持多种提取模式包括自定义索引、步长间隔和均匀分布，常用于帧采样和批处理优化。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | 必选 | IMAGE | - | - | 要提取的输入批量图像 |
| `mode` | - | COMBO[STRING] | step | index, step, uniform | 提取模式：index(自定义索引)、step(间隔采样)、uniform(均匀分布) |
| `index` | - | STRING | "0" | - | 自定义索引字符串，支持逗号分隔值和负数索引（如"0,2,4"或"1,-1"） |
| `step` | - | INT | 4 | 1-8192 | 步长模式的间隔值，从索引0开始每隔N张图像提取一张 |
| `uniform` | - | INT | 4 | 0-8192 | 均匀模式要提取的图像数量，在批次中均匀分布 |
| `max_keep` | - | INT | 10 | 0-8192 | 结果中保留的最大图像数量，防止内存使用过多 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 提取后的批量图像 |
