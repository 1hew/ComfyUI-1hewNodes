# Save Image - 保存图片

**节点功能：** `Save Image` 用于将 IMAGE 批次写入磁盘，并输出保存后的绝对路径。支持保存到 ComfyUI 输出目录或临时目录，并提供 UI 预览。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | 可选 | IMAGE | - | - | 需要保存的图像批次。 |
| `filename_prefix` | - | STRING | `image/ComfyUI` | - | 文件名前缀；遵循 ComfyUI 的保存路径规则。 |
| `save_output` | - | BOOLEAN | `true` | - | 保存到输出目录（true）或临时目录（false）。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `file_path` | STRING | 批量保存时以换行拼接的绝对路径列表。 |

## 功能说明

- 批量保存：对输入批次逐张落盘保存。
- 路径一致：使用与 ComfyUI 保存节点一致的命名与目录规则。
- 预览输出：在界面中返回已保存图像的预览结果。

## 典型用法

- 保存中间结果或最终结果，并将保存路径提供给下游文件类节点使用。

## 注意与建议

- `filename_prefix` 支持 ComfyUI 的占位符格式化能力。
