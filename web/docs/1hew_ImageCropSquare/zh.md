# Image Crop Square - 方形裁剪器

**节点功能：** `Image Crop Square` 将图像裁剪为以遮罩边界框为中心的正方形区域；当遮罩缺失或为空时，则退回为以图像中心为基准的方形裁剪。支持 `scale_factor`、额外留白，以及多种背景填充策略；可选 `apply_mask` 以遮罩透明度进行粘贴，并保证最终尺寸为 `divisible_by` 的倍数。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像批次 |
| `mask` | 可选 | MASK | - | - | 用于确定裁剪中心和大小的遮罩边界框；未连接或为空时使用中心裁剪 |
| `scale_factor` | - | FLOAT | 1.0 | 0.1-3.0 | 相对于 bbox 或中心方形的缩放系数 |
| `extra_padding` | - | INT | 0 | 0-512 | 在形成最终正方形前额外增加的留白 |
| `fill_color` | - | STRING | 1.0 | 灰度/HEX/RGB/名称/`edge`/`average`/`extend`/`mirror`/`mk`/`mask` | 背景填充策略，详见下方“填充策略” |
| `apply_mask` | - | BOOLEAN | false | - | 是否使用局部遮罩透明度将裁剪区域回贴到方形画布 |
| `divisible_by` | - | INT | 8 | 1-1024 | 保证最终方形宽高为该值的整数倍 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 裁剪后的方形图像批次；若批次内输出尺寸不同，会自动补齐到统一尺寸后堆叠 |

## 功能说明

- 遮罩感知方形：优先根据遮罩 bbox 计算方形区域；当遮罩为空时退回到图像中心方形裁剪。
- 尺寸控制：`scale_factor` 用于放大或缩小裁剪区域，`extra_padding` 增加对称留白，最终尺寸会对齐到 `divisible_by`。
- 填充策略：支持 `extend`（边缘复制）、`mirror`（镜像）、`edge`（四边平均色）、`average`（全图平均色），以及显式灰度/HEX/RGB/命名颜色；当 `apply_mask=True` 时，`mk`/`mask` 会使用遮罩区域平均色。
- 遮罩粘贴：启用 `apply_mask` 后，裁剪区域会使用局部遮罩进行柔和粘贴。
- 批量稳健性：如果同一批次输出尺寸不同，节点会自动补齐到公共尺寸再返回。

## 典型用法

- 人像方裁：提供人物遮罩，设置 `scale_factor≈1.2` 保留少量上下文，`fill_color=extend` 可获得自然边界。
- 中心方裁：不连接 `mask`，直接对图像中心做方形裁剪；使用 `average` 可获得较协调的背景色。
- 柔边拼贴：开启 `apply_mask`，利用遮罩透明度保留主体边缘过渡。

## 填充策略（`fill_color`）

- 灰度：如 `0.5` 表示中灰，会自动转为 RGB。
- HEX：如 `#FF0000` 或 `FF0000`。
- RGB：如 `255,0,0` 或 `0.5,0.2,0.8`（0-1 会自动转换到 0-255）。
- 颜色名称：如 `red`、`white`。
- `extend`：复制裁剪区域边缘像素向外填充。
- `mirror`：使用镜像方式向外填充。
- `edge`：使用裁剪区域四边的平均色分别进行填充。
- `average`：使用整张输入图像的平均色进行填充。
- `mk`/`mask`：在 `apply_mask` 启用时，使用遮罩区域的平均色作为填充色。

## 注意与建议

- 遮罩与图像尺寸不一致时，会先对遮罩进行居中对齐与画布调整，再计算 bbox。
- 若计算得到的 `final_size` 非正数，会回退到 `divisible_by`。
- `edge` 会按每条边分别采样平均色，减少角落像素对结果的偏置影响。
