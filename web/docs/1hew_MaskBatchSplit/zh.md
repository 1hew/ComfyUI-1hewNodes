# Mask Batch Split - 遮罩批次二段切分

**节点功能：** `Mask Batch Split` 将遮罩批次按数量切分为两段，可选择从开头或结尾取。提供边界处理并以非阻塞方式切片。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | - | MASK | - | - | 输入遮罩批次 |
| `take_count` | - | INT | 8 | 1-1024 | 取出的帧数量 |
| `from_start` | - | BOOLEAN | `False` | - | 为 `True` 表示从开头取，否则从末尾取 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask_1` | MASK | 切分后的第一段 |
| `mask_2` | MASK | 切分后的第二段 |

## 功能说明

- 方向控制：`from_start=True` 取前 `take_count`；否则取后 `take_count`。
- 边界处理：当 `take_count >= batch_size` 时，一侧输出为全部，另一侧为空。
- 异步切片：在工作线程执行，避免阻塞。

## 典型用法

- 将前缀遮罩与剩余遮罩分离输出。
- 保留最后 N 张遮罩并保留其余。
