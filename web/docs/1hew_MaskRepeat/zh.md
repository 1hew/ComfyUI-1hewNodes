# Mask Repeat - 遮罩重复

**节点功能：** `Mask Repeat` 节点用于将输入的遮罩（支持批量）进行指定次数的重复，并支持颜色反转功能。适用于需要将单个或一组遮罩扩展为更大批次的场景。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | 必选 | MASK | - | - | 输入遮罩批次。 |
| `count` | - | INT | 1 | 1-4096 | 重复次数。会将整个输入 Batch 视为一个单元进行重复。 |
| `invert` | - | BOOLEAN | False | True/False | 是否反转遮罩颜色（黑白互换）。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 重复（及可选反转）后的遮罩批次。 |

## 功能说明

- **批量重复**：
    - 输入 Mask Batch 为 `[A, B]`，`count` 为 2，输出为 `[A, B, A, B]`。
    - 整个输入 Batch 被视为一个整体进行复制。

- **反转功能**：
    - 当 `invert` 为 `True` 时，输出的遮罩颜色将反转（0 变为 1，1 变为 0）。
    - 此操作在重复之后进行（逻辑上等效，结果一致）。

## 典型用法

- **生成长序列遮罩**：将单个循环遮罩扩展为视频长度所需的帧数。
- **反转并扩展**：同时完成遮罩的反转和批量复制，简化工作流节点数量。
