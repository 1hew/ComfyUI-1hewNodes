# Image Tile Split Preset - 预设尺寸切片

**节点功能：** `Image Tile Split Preset` 按预定义尺寸将图像切片。支持用户选择固定尺寸或 `auto` 自动选择，以综合覆盖效率、宽高比匹配与切片数量进行优化。输出切片与用于无缝合并的丰富 `tile_meta`。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像；仅使用批次中的第一张。
