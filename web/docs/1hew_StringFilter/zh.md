# String Filter - 文本过滤器

**节点功能：** `String Filter` 对输入文本执行占位符替换、注释过滤与可选的空行移除，输出清洁的字符串。支持单行注释 `#` 与三引号多行块 `"""` / `'''`。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text` | - | STRING | `""` | - | 待处理文本，支持多行输入；会将字面 `\n` 转换为换行 |
| `filter_empty_line` | - | BOOLEAN | `false` | - | 是否移除空行 |
| `filter_comment` | - | BOOLEAN | `false` | - | 是否过滤注释（`#` 与三引号块） |
| `input` | - | STRING | `""` | - | 替换占位符 `{input}` 的字符串值 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `string` | STRING | 过滤后的文本字符串 |

## 功能说明

- 占位符替换：在处理前将 `{input}` 替换为输入值。
- 注释过滤：
  - 单行注释：`#` 及其后的内容被过滤。
  - 多行注释：支持 `""" ... """` 与 `''' ... '''` 块，未闭合时持续忽略直至闭合。
- 空行控制：可选移除空行；保留非空行末尾空白会被去除（右裁剪）。
- 换行规范化：字面 `\n` 会转换为真正的换行符。

## 典型用法

- 在拼接前对文本进行清洁，移除注释与空行，结合 `String Join Multi` 或 `Text List to String` 使用。
- 使用 `{input}` 将外部值注入到模板文本中，便于统一替换。

## 注意与建议

- 多行注释必须用成对三引号包围；在同一行闭合会被正确处理。
- 若仅需移除空行不处理注释，可将 `filter_comment` 置为 `false`。
