# Image Batch Uniform - 全批次均匀取图

**节点功能：** `Image Batch Uniform` 对整个图像批次做均匀采样，使用 `num_frame` 控制最终输出数量。`num_frame=0` 表示返回全部图像。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像批次 |
| `num_frame` | - | INT | 4 | 0-8192 | 最终输出数量；`0` 表示返回全部 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 均匀采样后的图像批次 |

## 功能说明

- `num_frame=0`：返回全部图像。
- `num_frame=1`：只取首帧。
- `num_frame=2`：取首尾两帧。
- `num_frame>=batch_size`：返回全部图像。
- 其他情况：在整个 batch 上做均匀抽样。
- 空批次安全：当输入为空时，输出为空批次。

## 典型用法

- 生成整段代表帧：设置 `num_frame=4`
- 只看首帧：设置 `num_frame=1`
- 只看首尾：设置 `num_frame=2`
- 保留全部：设置 `num_frame=0`

## 注意与建议

- 本节点始终基于整个 batch 均匀采样，不提供局部区间输入。
- 当 `num_frame` 小于总帧数时，会尽量均匀覆盖首尾与中间区域。
