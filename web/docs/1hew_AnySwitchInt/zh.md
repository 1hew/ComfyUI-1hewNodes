# Any Switch Int（多路整数切换）

`Any Switch Int` 支持动态输入端口与惰性求值。根据整数索引选择 `input_N` 的值，仅计算被选中的分支，适合复杂图中的多选项控制与路由。

## 输入

| 参数名称 | 必填 | 数据类型 | 默认值 | 取值范围 | 说明 |
| -------- | ---- | -------- | ------ | -------- | ---- |
| `select` | 是 | INT | 1 | 1–999999 | 选择 `input_N` 的索引（步长：1）。 |
| `input_N` | 否 | * | - | - | 动态输入端口：`input_1`、`input_2`、…。连接更多端口时，前端会自动扩展。支持任意类型。 |

## 输出

| 输出名称 | 数据类型 | 说明 |
|---------|----------|------|
| `output` | * | 直接传递选中输入的值；类型随选中输入而定。 |
