# Any Switch Int - 多路索引透传（1…10）

**节点功能：** `Any Switch Int` 通过整数索引在至多十个输入中选择其一并输出。实现惰性求值，仅计算被选中的输入端。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `select` | - | INT | 1 | 1–10 | 选择要输出的输入端索引 |
| `input_1…input_10` | 可选 | ANY (`*`) | - | - | 备选输入；仅选中的输入被计算（惰性） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `output` | ANY (`*`) | 选中的输入值 |

## 功能说明

- 选择范围：索引会限制在 `[1, SELECT_MAX]`，其中 `SELECT_MAX=10`。
- 惰性端口：仅选中的输入端会被计算。
- 选择健壮：当所选输入未提供时，输出为 `None`。

## 典型用法

- 模式复用器：用单一索引在多条子图中进行路由。
- 参数库选择：按索引用预设参数或资源。
- 结合空值判定：与 `Any Empty Int` 配合，从存在性生成索引。

## 注意与建议

- 提供与 `select` 对应的 `input_{k}`；未提供的索引将产生 `None` 输出。
- 将 `select` 作为安全的控制信号使用；越界索引会自动调整到有效范围。
