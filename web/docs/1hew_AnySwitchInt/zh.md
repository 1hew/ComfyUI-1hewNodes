# Any Switch Int - 多路整数切换

**节点功能：** `Any Switch Int`节点是一个多路整数切换器，支持多个输入选项的切换和惰性求值。根据整数索引（1-5）选择对应的输入输出，只计算选中的分支，提供灵活的多路数据路由能力。适用于多选项条件控制和复杂的数据流管理。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `select` | 必选 | INT | 1 | 1-5 | 选择输入的索引值，步长：1 |
| `input_1` | 可选 | * | - | - | 第1个输入选项（支持任意类型） |
| `input_2` | 可选 | * | - | - | 第2个输入选项（支持任意类型） |
| `input_3` | 可选 | * | - | - | 第3个输入选项（支持任意类型） |
| `input_4` | 可选 | * | - | - | 第4个输入选项（支持任意类型） |
| `input_5` | 可选 | * | - | - | 第5个输入选项（支持任意类型） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `output` | * | 根据索引选择的输出值（类型与选中的输入一致） |

## 功能说明

### 惰性求值机制

- **智能计算**：只计算选中索引对应的输入，其他输入不会被执行
