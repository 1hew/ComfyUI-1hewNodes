# Mask Fill Hole - 遮罩内孔填充

**节点功能：** `Mask Fill Hole` 对二值化遮罩进行内部孔洞填充，可选反转以得到反向结果。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | - | MASK | - | - | 输入遮罩批次；二维遮罩会扩展到 `[B,H,W]` |
| `invert_mask` | - | BOOLEAN | False | - | 为 `True` 时对填充结果取反 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 填充孔洞后的遮罩批次 |

## 功能说明

- 批量处理：逐项执行并控制并发。
- 孔洞填充：采用 `scipy.ndimage.binary_fill_holes` 并使用二维连通结构。
- 反转结果：可选对填充后的二值遮罩进行取反。

## 典型用法

- 分割清理：修复分割遮罩内部空洞以得到连续区域。
- 合成预备：保证区域连续性以便后续粘贴/混合操作。

## 注意与建议

- 内部以阈值 `>127` 二值化定义区域。
- 依赖 SciPy；请确保环境包含 `scipy.ndimage` 以进行孔洞填充。
