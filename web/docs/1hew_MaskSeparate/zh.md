# Mask Separate - 遮罩分离

**节点功能：** `Mask Separate` 将输入的遮罩根据连通区域进行分离，输出分离后的遮罩批次和数量。支持按面积过滤和多种排序模式。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | - | MASK | - | - | 输入的遮罩批次 |
| `threshold` | - | FLOAT | `0.5` | 0.0–1.0 | 二值化阈值 |
| `min_area` | - | INT | `1` | 1-100000000 | 最小连通区域面积，小于该面积的区域将被忽略 |
| `sort_mode` | - | COMBO | `top_to_bottom` | `top_to_bottom` / `left_to_right` / `area_desc` | 排序模式：从上到下、从左到右、面积降序 |
| `connectivity` | - | COMBO | `8` | `8` / `4` | 连通性：8 连通或 4 连通 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 分离后的遮罩批次 |
| `count` | INT | 分离出的遮罩数量 |

## 功能说明

- 连通域分析：基于二值化后的遮罩提取独立的连通区域。
- 面积过滤：通过 `min_area` 过滤掉过小的噪点或碎片。
- 排序输出：支持按空间位置（从上到下/从左到右）或面积大小（从大到小）对分离出的遮罩进行排序。
- 批量处理：对输入批次中的每一帧遮罩分别进行分离，并将所有结果合并为一个输出批次。

## 典型用法

- 实例分割：将包含多个物体的单一遮罩拆分为独立的物体遮罩。
- 噪点清理：结合 `min_area` 提取主要目标，去除细小噪点。
- 顺序处理：通过 `sort_mode` 确保分离出的遮罩具有稳定的顺序，便于后续按顺序进行处理。

## 注意与建议

- 阈值 `threshold` 决定了哪些像素被视为前景，通常保持默认值 0.5 即可。
- `connectivity` 决定了对角线相邻的像素是否被视为同一区域，8 连通更为宽松。
