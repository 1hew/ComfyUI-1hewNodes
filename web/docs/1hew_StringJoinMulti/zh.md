# String Join Multi - 多段文本拼接

**节点功能：** `String Join Multi` 接收最多 5 段文本，支持 `{input}` 占位符替换、注释过滤与空行控制，最终按分隔符拼接为一个字符串。分隔符支持转义与复合形式，如 `\n---\n`。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `text1` | - | STRING | `""` | - | 文本段 1，支持多行 |
| `text2` | - | STRING | `""` | - | 文本段 2，支持多行 |
| `text3` | - | STRING | `""` | - | 文本段 3，支持多行 |
| `text4` | - | STRING | `""` | - | 文本段 4，支持多行 |
| `text5` | - | STRING | `""` | - | 文本段 5，支持多行 |
| `filter_empty_line` | - | BOOLEAN | `false` | - | 是否移除空行 |
| `filter_comment` | - | BOOLEAN | `false` | - | 是否过滤注释（`#` 与三引号块） |
| `separator` | - | STRING | `"\n"` | 支持 `\n` / `\t` / `\r` / `\\` 组合 | 拼接分隔符，支持复合分隔符如 `\n---\n` |
| `input` | - | STRING | `""` | - | 用于替换 `{input}` 的字符串值 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `string` | STRING | 拼接后的字符串 |

## 功能说明

- 按段过滤：每段文本会独立执行占位符替换与注释/空行过滤，空白段自动跳过。
- 分隔符转义：支持 `\n`、`\t`、`\r`、`\\` 以及组合字符串（如 `\n---\n`）。
- 模板注入：统一通过 `{input}` 将外部值注入到各段文本。

## 典型用法

- 将多段提示词整理为单段输出，并在段间添加可视分隔：`separator=\n---\n`。
- 清除注释与空行后再拼接，得到更干净的上下文文本。

## 注意与建议

- 只有处理后非空的文本段才会参与最终拼接；如需保留空段，请关闭 `filter_empty_line`。 
- 多行注释块需成对三引号闭合；如需更复杂的过滤，请结合 `String Filter` 使用。
