# Image Resize Square - 方形尺寸适配

**节点功能：** `Image Resize Square` 提供通用方形尺寸预设，只保留 `256`、`512`、`1024`、`2048`、`4096` 五档，并支持 `auto` 与 `auto (0.5k | 1k)` 自动匹配模式。适用于需要将图像与遮罩统一调整为方形尺寸的流程。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `preset_size` | - | COMBO | `auto` | `auto` / `auto (0.5k \| 1k)` / `[0.25k] 256x256 (1:1)` / `[0.5k] 512x512 (1:1)` / `[1k] 1024x1024 (1:1)` / `[2k] 2048x2048 (1:1)` / `[4k] 4096x4096 (1:1)` | 方形目标尺寸选择；`auto (0.5k \| 1k)` 会优先在 `512` 与 `1024` 两档中自动匹配 |
| `fit` | - | COMBO | `crop` | `crop` / `pad` / `stretch` | 适应模式：裁剪、填充、拉伸 |
| `pad_color` | - | STRING | `1.0` | 灰度/HEX/RGB/颜色名/`edge`/`average`/`extend`/`mirror` | `pad` 模式背景填充策略 |
| `image` | 可选 | IMAGE | - | - | 输入图像批次 |
| `mask` | 可选 | MASK | - | - | 输入遮罩批次 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 调整后的图像批次 |
| `mask` | MASK | 与输出尺寸匹配的遮罩批次 |

## 功能说明

- 方形预设集：仅保留 `1:1` 尺寸，便于统一进入方图工作流。
- 自动匹配逻辑：`auto` 会在全部五档方形尺寸中按输入面积选择最近目标；`auto (0.5k | 1k)` 仅在 `512` 与 `1024` 中匹配。
- 适配策略一致：`crop` / `pad` / `stretch` 与现有 resize 节点行为一致，遮罩同步输出。
- 输入弹性：支持 image-only、mask-only、无输入三种场景。

## 典型用法

- 中小尺寸方图预处理：`preset_size=auto (0.5k | 1k)`。
- 需要统一到固定方图：直接选择 `256`、`512`、`1024`、`2048` 或 `4096`。
- 保留完整主体：`fit=pad`，并设置合适的 `pad_color`。

## 注意与建议

- 若追求更快的预处理速度，优先考虑 `512`、`1024` 或 `auto (0.5k | 1k)`。
- `auto` 会覆盖从 `256` 到 `4096` 的完整方形档位，更适合需要按输入大小自适应的流程。
