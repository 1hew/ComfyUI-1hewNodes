# Save Video - 视频保存与预览

**节点功能：** `Save Video` 将输入视频保存到输出目录，支持选择容器与编码器；当输入为空时直接完成并不产生副作用。保存后在界面提供预览入口。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `video` | 可选 | VIDEO | - | - | 待保存的视频；为空时节点直接通过。 |
| `filename_prefix` | - | STRING | `video/ComfyUI` | - | 保存路径前缀；支持由框架处理的格式占位符。 |
| `format` | - | COMBO | `auto` | 容器选项 | 容器格式选择；`auto` 自动选择合适格式。 |
| `codec` | - | COMBO | `auto` | 编码器选项 | 编码器选择；`auto` 根据容器与内容自动选择。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| - | - | 输出节点；返回保存视频的界面预览元素。 |

## 功能说明

- 空输入安全：当未提供 `video` 时直接完成。
- 适配批处理：依赖文件名中的自增 `counter` 避免同名冲突。

## 注意与建议

- 文件名模式为 `{filename}_{counter:05}_.{extension}`；通过设定 `filename_prefix` 引导目录与基本名。
- 启用元数据时，提示与额外信息会与视频一并保存。
