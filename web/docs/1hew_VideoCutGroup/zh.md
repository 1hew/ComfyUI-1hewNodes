# Video Cut Group - 视频硬切检测

**节点功能：** `Video Cut Group`节点用于检测视频中的场景切换点，通过分析相邻帧之间的相似度来识别硬切点，支持快速模式和精确模式，常用于视频分段和场景分析。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | 必选 | IMAGE | - | - | 输入的视频帧序列 |
| `threshold_base` | - | FLOAT | 0.8 | 0.0-1.0 | 基础阈值，用于判断帧间相似度的基准值 |
| `threshold_range` | - | FLOAT | 0.05 | 0.01-0.2 | 阈值范围，用于多阈值检测的浮动范围 |
| `threshold_count` | - | INT | 2 | 1-10 | 阈值数量，多阈值检测时使用的阈值个数 |
| `kernel` | - | STRING | "3, 7, 11" | 核配置 | 模糊核大小配置，逗号分隔的奇数值，如"3,7,11" |
| `min_frame_count` | - | INT | 10 | 1-1000 | 最小帧数，每个分段的最小帧数要求 |
| `max_frame_count` | - | INT | 0 | 0-10000 | 最大帧数，每个分段的最大帧数限制，0表示无限制 |
| `fast` | - | BOOLEAN | False | True/False | 快速模式，True为快速检测，False为精确检测 |
| `add_frame` | - | STRING | "" | 帧索引 | 手动添加切点，逗号分隔的帧索引，如"10,25,50" |
| `delete_frame` | - | STRING | "" | 帧索引 | 手动删除切点，逗号分隔的帧索引，如"15,30" |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 分组后的图像批次 |
| `group_total` | INT | 总分组数量 |
| `start_index` | INT | 每个分组的起始索引（列表） |
| `batch_count` | INT | 每个分组的帧数（列表） |
