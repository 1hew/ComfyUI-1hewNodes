# String Coordinate to BBoxes - 文本框坐标解析为列表

**节点功能：** `String Coordinate to BBoxes` 解析包含边界框的多行文本为嵌套列表结构，便于下游节点消费。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `coordinates_string` | - | STRING | `` | 多行 | 每行包含边界框坐标；分隔符可为空格或逗号；括号 `[]()` 将被忽略。每行至少含四个数 `x1 y1 x2 y2` |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `bboxes` | BBOXES | 嵌套列表结构，形如 `[ [ [x1,y1,x2,y2], ... ] ]`；当无有效框时返回 `[[]]` |

## 功能说明

- 健壮解析：移除括号，兼容逗号/空格分隔；将浮点转为整数；忽略无效片段。
- 结构兼容：输出嵌套列表，适配期望按组组织的下游消费者。

## 典型用法

- 每行一个框（如 `12,34,200,180`）；输出将所有行组织在同一个内层列表中。

## 注意与建议

- 当未解析到有效框时，输出为 `[[]]`。
