# Mask Stroke - 遮罩描边

**节点功能：** `Mask Stroke` 对输入遮罩生成外描边遮罩；支持在描边前先补洞、使用像素或百分比控制描边宽度，并可选择输出“描边环”或“描边与原遮罩并集”。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask` | - | MASK | - | - | 输入遮罩批次 |
| `fill_hole` | - | BOOLEAN | False | - | 为 `True` 时，先按 `Mask Fill Hole` 的规则补齐内部孔洞，再基于补洞后的遮罩生成描边 |
| `stroke_width` | - | FLOAT | 20.0 | 0–8192；`0<x<1` 视为百分比 | 描边宽度；小于 `1` 时按遮罩短边比例换算像素 |
| `include_mask` | - | BOOLEAN | True | - | 为 `True` 时输出描边与基础遮罩的并集；为 `False` 时仅输出描边环 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 生成的描边遮罩，或描边与基础遮罩的并集 |

## 功能说明

- 共享补洞事实源：`fill_hole=True` 时直接复用 `Mask Fill Hole` 的孔洞填充逻辑，避免结果漂移。
- 描边生成：使用椭圆结构元素对基础遮罩做膨胀，再减去基础遮罩，得到外描边区域。
- 百分比宽度：当 `stroke_width` 小于 `1` 时，按遮罩短边乘以该值换算为像素宽度。
- 批量处理：逐张遮罩并发处理，支持标准 `[B,H,W]` 批次输入。

## 典型用法

- 生成边缘引导：仅输出描边环，用作边缘约束或局部修补范围。
- 扩展可编辑区域：勾选 `include_mask`，得到“原区域 + 外扩描边”的组合遮罩。
- 修复空心区域：勾选 `fill_hole`，先补齐封闭孔洞，再统一生成稳定描边。

## 注意与建议

- `stroke_width=0` 时不会生成新描边；若 `include_mask=True`，输出为基础遮罩；否则输出为空遮罩。
- `fill_hole=True` 会将基础遮罩按孔洞填充规则二值化处理，适合封闭区域类遮罩。
