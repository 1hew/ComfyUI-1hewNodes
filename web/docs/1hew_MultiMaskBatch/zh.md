# Multi Mask Batch - 多掩码批次

**节点功能：** `Multi Mask Batch` 节点从动态 `mask_X` 输入构建掩码批次，以上一端口掩码尺寸为参照，按 `pad`、`crop`、`stretch` 统一尺寸，并使用双线性插值进行缩放。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `mask_1` | 可选 | MASK | - | - | 动态掩码输入首端口；可扩展 `mask_2`、`mask_3` 等 |
| `fit` | 必选 | COMBO[STRING] | pad | crop, pad, stretch | 掩码统一尺寸模式 |
| `pad_color` | 必选 | FLOAT | 0.0 | 0.0–1.0（步长 0.01） | `pad` 模式的填充值（灰度） |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `masks` | MASK | 统一后的掩码批次，数值范围钳制为 `[0, 1]` |
