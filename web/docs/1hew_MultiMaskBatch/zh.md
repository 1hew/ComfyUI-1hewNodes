# Multi Mask Batch - 多遮罩批次尺寸对齐

**节点功能：** `Multi Mask Batch` 将多张遮罩按第一张的尺寸进行对齐，支持 `crop`、`pad`、`stretch` 三种模式；在 `pad` 模式下使用常数值补边，并将对齐后的遮罩拼接为一个批次。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `fit` | - | COMBO | `pad` | `crop` / `pad` / `stretch` | 对齐模式 |
| `pad_color` | - | FLOAT | 0.0 | 0.0–1.0 | `pad` 模式下的补边常数 |
| `mask_1` | - | MASK | - | - | 第一遮罩；定义参考尺寸 |
| `mask_2…mask_N` | 可选 | MASK | - | - | 额外遮罩，按数字后缀排序识别 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `mask` | MASK | 对齐后的遮罩批次；裁剪到 `[0,1]` 浮点 |

## 功能说明

- 参考尺寸：使用 `mask_1` 的高宽作为目标。
- 适配模式：
- `stretch`：直接双线性缩放到目标。
- `crop`：缩放到覆盖目标后居中裁切。
- `pad`：缩放到适配目标内后按 `pad_color` 居中补边。
- 顺序采集：按 `mask_*` 的数字后缀收集并保序。

## 典型用法

- 批次统一：在逻辑运算或合成前统一遮罩尺寸。
- 语义区分：在 `pad` 模式下用常数值标记补边，以便后续处理区分内容与边界。
- 覆盖与保全：`crop` 保证覆盖内容，`pad` 保留完整遮罩内容。

## 注意与建议

- 补边以常数填充；请按下游语义选择合适的 `pad_color`。
- 输出为单通道遮罩，并沿批次维度进行拼接。
