# Image Resize QwenImage - 预设驱动的尺寸调整与适配模式

**节点功能：** `Image Resize QwenImage` 将图像与/或遮罩调整到为 QwenImage 工作流优化的预设分辨率。支持按输入长宽比自动选择预设（`auto`），三种适配模式（`crop`/`pad`/`stretch`），高级 `pad_color` 背景策略，并输出同步的遮罩。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `preset_size` | - | COMBO | `auto` | `auto` / `928×1664 [1:1.79]` / `1056×1584 [1:1.50] (2:3)` / `1140×1472 [1:1.29]` / `1328×1328 [1:1.00] (1:1)` / `1472×1140 [1.29:1]` / `1584×1056 [1.50:1] (3:2)` / `1664×928 [1.79:1]` | 目标分辨率预设 |
| `fit` | - | COMBO | `crop` | `crop` / `pad` / `stretch` | 调整模式 |
| `pad_color` | - | STRING | 1.0 | 灰度/HEX/RGB/颜色名/`edge`/`average`/`extend`/`mirror` | 在 `pad` 模式下的背景策略 |
| `image` | 可选 | IMAGE | - | - | 输入图像批次；可仅处理遮罩或生成纯色背景 |
| `mask` | 可选 | MASK | - | - | 输入遮罩批次；与输出尺寸同步 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 调整后的图像批次；无图像输入时为纯色背景 |
| `mask` | MASK | 按适配模式同步调整后的遮罩批次 |

## 功能说明

- 自动预设：依据图像或遮罩的长宽比选择最接近的预设；无输入时默认为 `1328×1328`。
- 所有输出裁剪到 `[0,1]` 浮点范围并保持设备一致性。
