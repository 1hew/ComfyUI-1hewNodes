# Int Split - 整数分割

**节点功能：** `Int Split`节点用于将总数值分割为两部分，支持百分比(0.0-1.0)和整数两种分割点输入方式。适用于需要按比例或固定数值分割整数的场景，如批处理大小分配、数据集划分等。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `total` | 必选 | INT | 20 | 1-10000 | 需要分割的总数值，步长：1 |
| `split_point` | 必选 | FLOAT | 0.5 | 0.0-10000.0 | 分割点，支持百分比(0.0-1.0)或整数值，步长：0.01 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `int_total` | INT | 原始总数值 |
| `int_split` | INT | 分割后的数值 |
