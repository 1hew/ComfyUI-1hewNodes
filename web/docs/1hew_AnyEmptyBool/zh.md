# Any Empty Bool - 统一的“空值”判断

**节点功能：** `Any Empty Bool` 判断输入是否为空，并输出布尔值。针对常见 Python 类型、张量、数组与嵌套容器提供一致的判定规则。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `any` | - | ANY (`*`) | - | - | 待判定的输入 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `bool` | BOOLEAN | 当输入为空时为 `True`，否则为 `False` |

## 功能说明

- 类型判定规则：
- `None` → 视为空。
- `str` → `len(value.strip()) == 0` 视为空。
- `bool` → `False` 视为空。
- `int/float` → 等于 `0` 视为空。
- `torch.Tensor` → `numel()==0` 或全部元素为 `0` 视为空。
- `numpy.ndarray` → `size==0` 或全部元素为 `0` 视为空。
- `list/tuple` → 长度为 `0` 或所有成员为空（递归判定）。
- `dict` → 长度为 `0` 视为空。
- 具有 `__len__` 的对象 → `len(obj)==0` 视为空。
- 异常保护：发生异常时返回 `True`（视为空）。

## 典型用法

- 流程分支：用于条件节点，根据数据是否存在启用/禁用子图。
- 参数兜底：结合 `Any Switch Bool`，在输入为空时走默认分支。
- 批处理安全：在下游处理前确认数组/张量并非全零。

## 注意与建议

- 张量/数组需全部为零才视为空；存在任一非零元素则视为非空。
- 嵌套容器按递归逻辑判定，可对复合输入进行深度检查。
