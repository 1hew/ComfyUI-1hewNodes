# Any Empty Bool - 通用空值检查（布尔输出）

**节点功能：** `Any Empty Bool`节点用于检查任意类型输入是否为空，返回布尔值结果。支持多种数据类型的空值判断，包括字符串、数值、张量、列表、字典等，为条件逻辑控制提供通用的空值检测能力。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `any` | 必选 | * | - | - | 需要检查的任意类型输入值 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `bool` | BOOLEAN | 空值检查结果，True表示为空，False表示非空 |
