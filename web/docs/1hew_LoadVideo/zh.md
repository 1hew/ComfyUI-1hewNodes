# Load Video - 加载视频对象

**节点功能：** `Load Video` 用于选择视频文件并输出 VIDEO 对象，供下游节点消费。支持目录扫描、索引选择，并在解码阶段应用帧参数设置。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `path` | - | STRING | `""` | - | 视频文件路径或目录路径。 |
| `frame_limit` | - | INT | `0` | 0-100000 | 输出帧数上限；`0` 表示保留设置后的全部帧。 |
| `fps` | - | FLOAT | `0.0` | 0-120 | 目标 FPS，用于重采样；`0` 表示沿用源 FPS。 |
| `start_skip` | - | INT | `0` | 0-100000 | 起始跳过帧数。 |
| `end_skip` | - | INT | `0` | 0-100000 | 末尾跳过帧数。 |
| `format` | - | COMBO | `4n+1` | `n` / `2n+1` / `4n+1` / `6n+1` / `8n+1` | 帧数格式约束，重采样后应用。 |
| `video_index` | - | INT | `0` | -8192-8192 | `path` 为目录时的视频索引；支持负数索引（按数量取模）。 |
| `include_subdir` | - | BOOLEAN | `false` | - | 目录扫描时包含子目录。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `video` | VIDEO | 视频对象，按给定参数解码并提供图像帧与音频。 |

## 功能说明

- 文件/目录输入：支持单视频文件与目录视频集合选择。
- 索引选择：路径稳定排序与取模索引，便于循环遍历。
- 帧参数：依次应用跳帧、FPS 重采样、格式约束、帧数上限。
- Alpha 解码：在源流支持时优先解码为 RGBA。
- 音频支持：存在音频流时解析并随视频对象一并输出。

## 典型用法

- 从目录按索引选择视频，直接连接 `Save Video` 做复制保存与预览。
- 在转帧前对片段进行裁切与 FPS 统一，再连接到 `Load Video to Image` 或其它处理节点。

## 注意与建议

- 当容器解码链路需要增强时，会使用 ffmpeg 作为帧解码补充能力。
