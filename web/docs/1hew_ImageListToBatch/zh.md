# Image List to Batch - 列表合并为填充批次

**节点功能：** `Image List to Batch` 将图像列表合并为单个批次。统一为末通道排列并按最大高宽进行零值填充，使得可安全拼接。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image_list` | 列表 | IMAGE/LIST | - | - | 图像列表或单张图像 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image_batch` | IMAGE | 经过零值填充到公共尺寸后的批次（`B×H×W×C`） |

## 功能说明

- 输入灵活：接受单张或列表/元组；非张量元素自动忽略。
- 形状统一：将 `H×W×C` 转为 `1×H×W×C` 以便拼接。
- 尺寸对齐：对每张图像按所有输入的最大高宽进行零值（黑色）填充。
- 回退策略：当无有效图像时返回空批次 `(0, 64, 64, 3)`；仅 1 张图像时直接返回该图像。

## 典型用法

- 将不同尺寸的帧合并为批次供模型处理。
- 合并多分支结果为一个连续批次。

## 注意与建议

- 填充采用常数 `0`，并在拼接前保持设备与数据类型一致。
