# Detect Guide Line - 引导线检测

**节点功能：** `Detect Guide Line` 通过边缘检测、直线提取、交点聚类与分位数筛选，检测透视引导线并估计消失点。输出带标注的原图、仅线条的叠加图，以及引导线的二值遮罩。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | 必选 | IMAGE | - | - | 输入图像张量 |
| `canny_low` | 必选 | FLOAT | 0.2 | 0.0–1.0 | Canny 低阈值（映射到 0–255） |
| `canny_high` | 必选 | FLOAT | 0.8 | 0.0–1.0 | Canny 高阈值（映射到 0–255，且不低于低阈值） |
| `seg_min_len` | 必选 | INT | 40 | 1–300 | `HoughLinesP` 的最小线段长度 |
| `seg_max_gap` | 必选 | INT | 8 | 1–100 | 线段允许的最大间隙 |
| `guide_filter` | 必选 | FLOAT | 0.6 | 0.1–1.0 | 线条筛选强度；数值越大保留的线条越多（分位阈值 = `100 - guide_filter*60`） |
| `guide_width` | 必选 | INT | 2 | 1–100 | 引导线与消失点标记的绘制宽度 |
| `cluster_eps` | 必选 | INT | 30 | 1–100 | DBSCAN 聚类半径（像素）用于交点聚类 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `image` | IMAGE | 绘制到消失点的引导线标注图 |
| `line_image` | IMAGE | 仅线条图（黑底红线，含消失点标记） |
| `line_mask` | MASK | 引导线二值遮罩（线条处为 255，其余为 0） |
