# Text Compare - 文本比较

**节点功能：** `Text Compare` 对两个输入字符串执行比较操作，输出布尔结果。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `a` | - | STRING | `""` | - | 左侧输入字符串。 |
| `operator` | - | COMBO | `==` | `==` / `!=` / `⊂` / `⊃` / `⊄` / `⊅` / `startswith` / `endswith` / `regex` | 比较运算符。 |
| `b` | - | STRING | `""` | - | 右侧输入字符串。 |

## 输出

| 输出名称 | 数据类型 | 描述 |
|---------|----------|------|
| `bool` | BOOLEAN | 比较结果布尔值。 |

## 运算符说明

| 运算符 | 含义 | 说明 |
|--------|------|------|
| `==` | 相等 | `a` 等于 `b` |
| `!=` | 不等 | `a` 不等于 `b` |
| `⊂` | 包含于 | `a` 是 `b` 的子串 |
| `⊃` | 包含 | `b` 是 `a` 的子串 |
| `⊄` | 不包含于 | `a` 不是 `b` 的子串 |
| `⊅` | 不包含 | `b` 不是 `a` 的子串 |
| `startswith` | 以…开头 | `b` 以 `a` 开头 |
| `endswith` | 以…结尾 | `b` 以 `a` 结尾 |
| `regex` | 正则匹配 | 在 `a` 中搜索正则表达式 `b` |

## 典型用法

- 判断两个字符串是否相同：`operator="=="`，比较两个前缀/后缀节点的输出。
- 检查提示词中是否包含关键词：`operator="⊂"`，`a` 为关键词，`b` 为提示词文本。
- 正则提取判断：`operator="regex"`，`b` 为正则表达式，`a` 为待匹配文本。

## 正则表达式常用示例

| 场景 | `b` 正则 | `a` 示例 | 说明 |
|------|----------|-----------|------|
| 是否纯数字 | `^\d+$` | `12345` | 匹配至少一位数字 |
| 是否纯英文 | `^[A-Za-z]+$` | `Hello` | 仅匹配英文字母 |
| 是否包含中文 | `[\u4e00-\u9fff]` | `你好世界` | 匹配任意中文字符 |
| 是否图片文件名 | `(?i)\.(png\|jpe?g\|gif\|webp)$` | `photo.jpg` | 忽略大小写匹配图片后缀 |
| 是否视频文件名 | `(?i)\.(mp4\|webm\|mov\|avi)$` | `movie.mp4` | 匹配常见视频后缀 |
| 提取文件名前缀 | `^(.+?)(?:_v\d+)?$` | `image_v3.jpg` | 捕获去除版本号前后的文本 |
| 是否种子值 | `^\d{1,10}$` | `4294967295` | 1~10 位数字 |
| 是否负面提示埋点 | `NSFW\|nude\|naked` | `photo of people` | 用 `\|` 分隔多关键词正则 |
| 是否长宽比字符串 | `^\d+:\d+$` | `16:9` | 匹配 `n:n` 格式 |
| 是否含 lora 权重 | `:\d+(\.\d+)?>` | `<lora:name:0.8>` | 检测 lora 触发语法 |

## 注意与建议

- `startswith` / `endswith` 中 `a` 是匹配片段，`b` 是被匹配的字符串。
- `regex` 模式下若正则表达式无效，返回 `False`。
- 所有输入在比较前会被转换为字符串。
