# Image Grid Split - 图片宫格分割器

**节点功能：** `Image Grid Split` 将图像按 `rows × columns` 宫格分割，并根据 `output_index` 输出全部子图批次或每张图的单个子图。

## 输入

| 参数名称 | 入端选择 | 数据类型 | 默认值 | 取值范围 | 描述 |
| -------- | -------- | -------- | ------ | -------- | ---- |
| `image` | - | IMAGE | - | - | 输入图像批次 |
| `rows` | - | INT | 2 | 1–10 | 分割行数 |
| `columns` | - | INT | 2 | 1–10 | 分割列数 |
| `output_index` | - | INT | 0 | 0–100 | 0：输出所有子图；>0：每张图输出第 N 个子图。

## 典型用法

- Patch 推理：将大图拆分为网格子块以进行模型推理；设 `output_index=0` 输出全部。
- 定位子块：当仅需每张图某个区域时，设定 `output_index` 为该子图索引。

## 注意与建议

- 为获得严格对齐，建议确保图像尺寸可被 `rows` 与 `columns` 整除；否则每轴末尾像素会因整数除法被舍去。
- RGBA 输入会在内部转换为 RGB 子图以统一形状。
