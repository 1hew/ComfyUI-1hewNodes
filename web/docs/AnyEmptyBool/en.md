# Any Empty Bool

**Node Function:** The `Any Empty Bool` node checks whether any type of input is empty and returns a boolean result. It supports empty value detection for various data types including strings, numbers, tensors, lists, dictionaries, providing universal empty detection capabilities for conditional logic control.

## Inputs

| Parameter | Required | Data Type | Default | Range | Description |
|--|--|--|--|--|--|
| `any` | Required | * | - | - | Input value of any type to be checked |

## Outputs

| Output Name | Data Type | Description |
|-------------|-----------|-------------|
| `bool` | BOOLEAN | Empty check result, True indicates empty, False indicates non-empty |
